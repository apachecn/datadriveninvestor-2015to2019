以下是系统工程中的四种主要模型类型。如果你首先为你的设计选择最好的系统，你将能够更好地向投资者解释你的原型。

**分析模型** —当您有一个描述您正在研究的行为的数学模型或方程时，这种模型类型最适合。估计季节性活动期间店面的平均排队长度就是一个例子。第二个例子是当你在计算弯曲一根钢筋所需的力时。两种情况都使用等式。

**模拟模型** —蒙特卡洛模型是模拟模型的经典例子。当您有可变性并对预期行为做出假设时，请使用此模型类型。模拟模型通常会反复运行多次，以得出描述结果的平均值。当你没有一个函数方程和有可变性时，使用模拟方法。

**物理模型**——当你需要了解你的设计如何适应一个空间时，使用物理原型。当你没有一个真正的产品时，你必须想出一些可能有用也可能没用的点子。你怎么能测试或模拟一个还不存在的东西呢？一个例子是[格鲁曼飞机公司](https://en.wikipedia.org/wiki/Apollo_Lunar_Module)为阿波罗登月任务建造登月舱。

**实际系统测试或实际模型** —当您需要真实数据和结果时，请使用它。一个例子可能是火箭发动机测试。另一个例子是培训新员工使用安装在办公室的软件工具。
