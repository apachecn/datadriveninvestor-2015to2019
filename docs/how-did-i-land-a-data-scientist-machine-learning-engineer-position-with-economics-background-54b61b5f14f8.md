# **我的经济学背景以及与数据科学的相似性**

![](img/8a5fafa3363fa5478ea21b50ac0e3950.png)

Photo by [Ben White](https://unsplash.com/@benwhitephotography?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

那么，数据科学家到底在做什么呢？

一般来说，成为一名数据科学家需要具备 3 个要素。

1.  数学和统计技能
2.  编程(软件工程)技能
3.  领域知识

![](img/e217f4740966580dfa97c7df8970f685.png)

Components of Data Science ([https://www.eventshigh.com/detail/Hyderabad/328bc517f3413994deb4739eb34d451e-data-science-how-to-start?src=widget&widget=true&weekwid=1&widHeader=1](https://www.eventshigh.com/detail/Hyderabad/328bc517f3413994deb4739eb34d451e-data-science-how-to-start?src=widget&widget=true&weekwid=1&widHeader=1))

**数学与统计**

我的专业是经济学，无论是学士还是硕士，我相信如果你完成了经济学课程直到硕士，你已经有很好的潜力成为一名数据科学家。

大多数经济学学士课程提供基本的数学和统计学(计量经济学)课程，在我的课程中，有线性代数，微积分，概率论和大量的统计学和计量经济学课程，包括像 R 和 SPSS 这样的统计软件。硕士课程提供高级优化问题和各种统计方法。所以实际上经济学课程确实有助于成为一名数据科学家，至少在数学和统计技能方面。

**编程(软件工程)**

对于经济学学生/毕业生来说，棘手的部分在于编程(软件工程)技能。通常，没有多少经济学学生在他们的课程中学习编程。他们中的大多数人甚至认为这是一件特别和可怕的事情。然而，例如，Python 是数据科学和机器学习最常用的语言，它是一种面向对象的语言，非常容易掌握。此外，R 应该是计量经济学专业的学生非常熟悉的，因为它是一种统计语言。

我所在的大学开设了一门名为“科学家 Python”的 Python 课程，这是为期两周的 Python 夏令营，你从早上 9 点到下午 5 点练习编程(是的，这是德国式的)。在夏令营结束时，我们大多数人都可以编程，例如使用蒙特卡罗模拟来估计恒星的大小。这是非常激烈的，但几天后你会觉得使用 Python 很舒服。

如果你能学习软件工程技能，比如 cloud、Docker、Kubernetes 和产品开发，那就更好了。但这些主要是针对机器学习工程师的(我现在也在学)。

**领域知识**

通常，人们在开始工作后才学习与业务相关的知识。然而，由于经济学涉及广泛的话题，*我想*我们对任何种类的概念都有一定的灵活性。

# **面试经历以及我是如何度过的**

![](img/027e342623f2cdb5413bf8e5b83e998e.png)

Photo by [Icons8 team](https://unsplash.com/@icons8?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

那么我最后是怎么得到机器学习工程师这个职位的呢？

第一，在我毕业前 1.5 年左右，我想做一名咨询师，用统计数据解决所有的商业问题。于是申请了东京四大的顾问职位，拿到了 offer。然而，我觉得那个职位对我来说有些东西是不够的，我想以科学的方式做，所以申请了东京一家技术咨询公司的数据科学家职位。这些面试非常标准，没有编程测试。

在我得到一个数据科学家职位的 offer 后，我想提高我的机器学习技能。我在大学里学习了机器学习课程，最后做了我的第一个机器学习项目作为我的硕士论文([利用机器学习技术分析网上价格](https://www.researchgate.net/publication/324749650_Analyzing_online_price_by_using_machine_learning_techniques?_sg=vOsx8X_-YTdJ_TVz3rHngewQr_FPKKENXvqSl4AD7GRxoAcl8fFUmjfA4Z88jByvOiZh6Dp2TRAMkAKR30FBz72qYJRI8Ce-7zE8zJzJ.07BVPeeiQvMw9motmArWoQCGTFlE1WH-MzPqzkPsSY_5nttWmcUXjMuH5MjjPi9CThGUqWA229NG4IxppK3jtQ))。我将在下一篇文章中详细讲述这篇论文。

然而，我改变了主意，我想留在欧洲。我开始申请职位。我申请了很多职位，主要是在欧洲的大城市，包括伦敦。我得到了不少面试机会。他们起初质疑我为什么不管我的经济学背景都要申请数据科学/机器学习工程师的职位，但似乎喜欢我的机器学习项目，它展示了扎实的数学、统计知识和整个机器学习生命周期的经验(除了部署)。

面试挺有意思的，有概率，有编码，有机器学习知识测试。如果你做了功课(复习基本概念及其应用)，它们不是不可能通过的。

最终，我决定接受目前在伦敦的公司的邀请。

这基本上就是我如何从经济学背景中获得机器学习工程师职位的。

# **成为数据科学家的秘诀**

![](img/1094adbc79c6e842fbe80f9bf989bae4.png)

Photo by [Júnior Ferreira](https://unsplash.com/@juniorferreir_?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

1.  **提高你的编程技能**

如果你从经济学专业毕业，你很可能擅长数学和统计学。要成为一名数据科学家，你唯一能做的就是编程。建议在你的大学或者 Coursera 或者任何 MOOC 课程上一些结构化的课程。有一些很好的课程，比如 IBM 的数据科学 Python 和 Coursera 的 AI T1。

**2。制作自己的机器学习项目**

你可以将机器学习应用于几乎任何类型的程序(嗯，这可能是数据是否可用的问题)。并在 GitHub 上创建您的存储库(我肯定应该这么做)。在 kaggle 比赛中工作也是一个很好的练习，但是考虑到最近比赛的复杂性，这可能不是最好的方法。

**3。参加数据科学会议**

伦敦有许多有趣的聚会，我经常参加。在那里，你可以学到一些有用的东西，也可以和其他数据科学家交流，分享一些想法和项目。如果你所在的城市没有 meetup，不要担心，也有很多网络研讨会。让你了解这个行业的新话题是非常重要的。因此 meetup 在这方面也非常有用。
