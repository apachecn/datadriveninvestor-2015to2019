在决定从近 600 种可用于生产的语言中的哪一种语言开始之前，必须有一个具体的目标。之后，根据你的目标过滤可用的语言。不是每种编程语言都与任何平台兼容。这意味着有些是专门为 web 应用程序、移动应用程序和桌面应用程序设计的。如果你主要关心的是移动应用，选择一种相关的语言。

编程语言永远不会停止发展，因此在选择编程语言时，考虑它的受欢迎程度是至关重要的。流行语言有据可查。因此，当你遇到困难时，找个人帮你就容易多了，而且一种语言的受欢迎程度也随着它的市场需求而变化。

如果你刚刚开始，这里有一个流行的和初学者友好的语言列表，你可以从中选择。

*   Javascript(特定于 web 应用程序)
*   Java 语言(一种计算机语言，尤用于创建网站)
*   计算机编程语言
*   C
*   C++
*   C#
*   PHP(网络应用程序)
*   红宝石
*   Swift(特定于 Mac OS 和 IOS 应用程序)
*   去

上述语言在就业市场上有很高的需求，因为它们大多被大型软件开发行业使用，并且它们的受欢迎程度和文档化程度更高。

既然我们已经接近这篇文章的结尾，如果你对这些编程语言有热情，我建议你去尝试一下。此外，当你在练习中选择一门不令人愉快的语言时，你很容易放弃它。因此，请确保您根据其他开发者或搜索结果的启发选择了自己喜欢的语言。
