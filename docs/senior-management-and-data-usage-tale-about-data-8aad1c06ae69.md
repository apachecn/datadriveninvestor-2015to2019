如果你是一个高级管理人员，这篇文章是给你的。在这篇文章结束时，我希望你会相信，是时候向你的商务智能主管提出要求，甚至要求做出改变了。

在过去的一年里，我和许多高层管理者交谈过，我发现他们都有两个共同点；他们没有太多的时间，而且，他们通常无法直接访问自己部门或公司的数据。这降低了他们做出决策和为业务的未来发展而努力的能力。许多经理告诉我，他们通常等待下属为他们提供问题的答案，这是让数据具有可操作性的一大障碍。2019 年，事情要变了！

您组织的 BI 团队或分析团队的工作首先是让高级管理人员能够方便地使用和研究来自其所有下属部门的数据。他们应该构建报告，在更精细的层次上关注部门的活动和绩效。这将允许高级管理人员深入研究数据，了解什么可行，什么不可行，而不需要等待别人传达这一信息。它应该在他们的指尖可以接触到并且不被过滤。我并不是建议你应该进行微观管理，而是应该利用这些数据来寻找新的机会，以改进策略、用户体验和跨部门的总体方向。

拥有你的数据和使用你的数据不是不同的概念。它们都允许您在一个真正的数据源中维护和收集内部原始数据。将所有这些信息收集到一个仓库中，使您和您的员工能够使用他们喜欢的任何工具来访问这些信息。它还能确保你们看到并分析相同的数据，最重要的是，确保你们在制定战略和计划时使用最好的可用数据。

例如，让您能够使用 Tableau 或 Power BI 轻松访问数据不需要您太多的学习。如果数据集构建正确，这些工具中的大多数都将是直观且易于导航的。经过 3 个小时的培训(主要针对数据集)，您将能够构建自己的报告和仪表盘，在快速响应的环境中回答您的所有问题。与带有透视数字的 Excel 表格相比，这是多么大的变化啊。

我非常相信“从 c 级到初级的一个报告”。该报告帮助组织的所有级别使用数据来调整和理解业务目标。然而，这只能作为我们所处位置以及我们离实现目标还有多远的一个指标。但作为一名高级经理，你应该主要关注探索未来的新机遇。

现在，你所需要做的就是致力于做更多的数据研究，并取得更好的结果。这就是你创造成功和快速发现新趋势的方法。
