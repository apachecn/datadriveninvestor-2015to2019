# 数字字节区块链每周综述

乔尼·弗莱

# 控制叙事

阿德里安·波特

人类喜欢从虚无中制造意义。我们很自然地赋予常见事件内涵，并从这种潜台词中塑造我们的行为。我们通常在活动开始时立即这样做。我们急于对某事是好是坏下结论。从那以后，一旦我们分配了标签，我们就要花很多时间来更改或删除它。
