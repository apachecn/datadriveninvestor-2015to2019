机器学习是计算机科学的一个领域，在这个领域中，程序被开发成能够自我学习，而不需要定义特定的输出。简单地说，它是一种处理先前信息并根据这些经验创造新产出的方法。机器学习的一个很好的例子就是我们邮箱里的垃圾邮件过滤器。对于电子邮件提供商来说，定义哪些电子邮件是垃圾邮件以及哪些与每个人相关是非常困难和耗时的。因此，计算机算法通过使用我们和网络其他用户提供的数据，学习哪些电子邮件可以忽略，哪些应该直接进入我们的邮箱。因为这是一个学习过程，有时我们会在垃圾邮件箱中发现一些不应该在那里的电子邮件，然后我们通知这不是垃圾邮件，程序会在下次纠正它。

机器学习是现在非常流行的人工智能(AI)的一部分，人工智能通过承担以前只有人类才能完成的任务，如驾驶汽车，正在彻底改变世界。在这种情况下，机器(汽车)通过使用传感器和不同来源的数据进行学习，例如，交通规则、关于拥堵的数据和用户偏好。与我们驾驶时的视觉相比，用于距离测量的激光和雨水探测器等传感器对于向汽车提供有关驾驶条件的信息非常重要。我们从经验中得知，在下雨的时候，在我们附近有太多汽车的时候，我们应该更加小心，这同样适用于自动驾驶汽车。

机器学习算法处理大量数据，在某些情况下，需要非常高的计算能力。然而，信息技术世界发展如此之快，其容量几乎每隔几年就翻一番，这将为我们当今面临的复杂问题创造解决方案。

每个机器学习程序都使用统计分析来评估收集的数据，并定义下一步要做什么。通过处理这些数据，程序创建了一个关于主题的概括，并使用它在新的或不可预见的情况下采取行动。一个例子是，当我们看到一只蜘蛛时，我们决定逃跑或杀死它。我们通过经验(父母、书籍、文章、自己的经历)了解蜘蛛可能危险，也可能不危险。通常，我们会考虑蜘蛛的大小、外貌、蜘蛛网的延伸、颜色，并基于此做出决定:逃跑或杀死它(或者如果它不在我们的房子里，就让它呆着)。机器学习程序也可以基于传感器(与我们有相似的感知)和经验(与我们所学的相似)来学习做类似的动作。

虽然很难预测机器学习程序和人工智能能给世界带来多大的变化，但可以肯定的是，它们在许多情况下都能胜过人类。以车祸死亡人数为例。如果我们有自动驾驶汽车，他们至少不会酒后驾车或不遵守速度限制。当这些新技术起飞时，许多其他领域都可以得到改善。

看到更多这样的事情成为现实只是时间问题。
