你最近看过你的日历吗？我指的是 Outlook、Gmail、iPhone(或任何其他手机)应用程序或你使用的任何应用程序中的日历。

如果你像大多数其他人一样，它可能充满了连续的会议。如果你是一个丰富多彩的人，可能会有多个不同颜色的会议类别。

我想问你的问题是，你安排了多少次会议只是为了和自己交谈？

> “每天至少要和自己说一次话，否则你可能会错过和这个世界上一个优秀的人的相遇。”斯瓦米·维威卡难达

## 在日历上为自己预定会议

我从我的一位首席信息官那里学到的一点是，在未来的几个月里，每周至少有几天为自己留出几个小时的时间。

许多年前，当我管理 ERP 支持团队时(如果你在 IT 支持部门工作过，你会理解有时会有多疯狂)，我常常被问题的每一个细节所困扰，没有时间去解决问题。

## 如果你不做，别人会做的

如果日历是空的，你不为自己做，别人也会开始屏蔽。

思考时间比花时间和做会议记录更重要。

当我们计划度假时，我们都擅长阻止日历和设置外出通知——这太提前了。但这并不适用于每天与最重要的人——我们自己——交谈。

> 你安排了多少次会议只是为了和自己交谈？

## 成为思想领袖

如果你想成为一名思想领袖，那就提前安排时间来回顾这些想法、未来计划和活动。

你会开始注意到，人们会试图在那些被屏蔽的时段安排他们想要的任何会议。

有时，他们可能会找你，询问你是否可以移动这些会议(他们不知道这是一个安排了时间的会议)，然后你可以回复是否真的有重要的事情要参加。

## 利益

1.  你开始控制你的时间
2.  你开始把时间花在真正重要的活动上
3.  你可以以更好的方式优先处理你的任务
4.  你开始更多地了解自己
5.  你开始得到直升机视图或鸟瞰图
6.  你(和你的团队)的生产力提高了
