# 在反思 2018 年之前，不要急于进入 2019 年

> 原文：<https://medium.datadriveninvestor.com/dont-rush-into-2019-before-reflecting-on-2018-68ea9f8b84cd?source=collection_archive---------10----------------------->

[![](img/03d8f1aeebc41484ea506e8ca31e65fd.png)](http://www.track.datadriveninvestor.com/1B9E)![](img/d17c3b76a2877e007a692abb536d496b.png)

我讨厌新年决心。为什么？因为它们通常是宽泛的、总括性的陈述，几乎没有责任或基于现实。它们通常是基于我们不高兴的事情或在过去一年中做错的事情，而没有真正反思学到的教训或理解为什么我们没有完成一些事情(锻炼、更有耐心、辞掉让你痛不欲生的工作)。

我相信的是设定目标。有明确的目标，有可衡量的目标，你可以在这一年努力实现。

但在你设定目标之前，你需要掀开被子，诚实地看待 2018 年。扪心自问:

*   在你的个人生活和职业生涯中，你最自豪的成就是什么？
*   你最大的胜利是什么？
*   你在哪里胡茬，或所有的崩溃？
*   谁是你最大的盟友？
*   你最大的缺点是什么？
*   谁帮助你或者可以帮助你弥补这些弱点(帮助你成长或者你可以把工作交给他们，因为这是他们的强项)？
*   你在哪里获得了最大的成长？
*   你还需要在哪些方面努力？

这些只是你在回顾 2018 年时要问自己的一些问题。

对我来说，我辞掉了一份工作，这样我就可以专注于写作、公共关系咨询和企业培训。这是一个巨大的飞跃，从有福利的定期薪水到建立业务、客户基础和学习在线营销的世界(似乎每周都在变化)。

我在反思 2017 年后做出了这一跳跃，并意识到我在太多的方向上伸展，并且没有在日常工作中追随我的激情。我做的工作是解剖 2017 年，设定 2018 年的目标。在花了一些时间把事情安排妥当后，我又回到了自我创业的世界。(让我们明确一点，再多的计划也无法实现平稳过渡。这就是为什么它叫做跳跃。)

虽然我们在 2018 年只剩下几个小时了，而且你们中的许多人要到 2019 年才会读到这篇文章，但我鼓励你们在未来几天的日历上预订时间，花几个小时真正反思过去的一年。你可以单独做这件事，可以和朋友一起做，也可以和你的伴侣一起做。你怎么做并不重要。重要的是你花时间去反思。诚实(这一年没有糖衣，而是寻找成长和学习的方法)。

你猜怎么着？虽然你可能会发现一些丑陋的事实，但你也会看到你已经走了多远。2018 年有一些成就可以让你感到骄傲和庆祝。沉浸在成功中，这样你才有精力在 2019 年的增长领域开展工作。

这种反思将有助于让 2019 年成为不可思议的一年！！！
