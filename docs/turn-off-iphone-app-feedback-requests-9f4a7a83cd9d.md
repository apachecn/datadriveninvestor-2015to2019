# 应用内评级和评论

要关闭这些应用内反馈弹出窗口，请在 iPhone 上执行以下步骤:

*   设置
*   iTunes 和 App Store
*   关闭“应用内评分和评论”

禁用此功能将阻止任何 Apple 赞助的试图收集用户反馈的反馈弹出窗口。不幸的是，一些应用程序抓住了这种偏好，并一直在开发变通方法，例如将反馈表单直接集成到应用程序 UI 中。

![](img/75b4a1efdb1a851c8749c7535e5a57f7.png)

由于这种禁用弹出功能相对较新，苹果正在积极谴责任何试图绕过用户偏好的应用程序。无论如何，禁用“应用内评级和评论”仍然有助于整理你的移动应用体验。我在自己的 iPhone 上关闭了这个功能，并立即注意到了一个改进。
