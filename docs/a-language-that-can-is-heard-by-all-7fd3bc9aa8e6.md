# 什么是 Babel，它将如何帮助你编写 JavaScript？

Babel 是一个 JavaScript transpiler，它将 edge JavaScript 转换成普通的旧版本 ES5 JavaScript，可以在任何浏览器上运行(甚至是旧版本)。
