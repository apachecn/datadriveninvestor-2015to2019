# 想留住顶尖人才？成为顶级公司

## 企业基于工作场所文化使自己与众不同

![](img/5844c5e673dcbbada63510f615bf5a43.png)

随着国家基本上处于充分就业状态，公司必须全力以赴[招募顶尖人才。企业需要阻止员工加入竞争对手，并确保新员工增加价值，让公司发展壮大。](https://www.gnapartners.com/lp/recruiting-rpo/)

[艾丽莎·马丁](https://twitter.com/EleeshaMartin)和[莎拉·布伦南](https://twitter.com/ImSoSarah)谈论了公司吸引顶级求职者的招聘策略和步骤。

Martin 是一家人力资源和行政外包公司的高级招聘专家，该公司提供人力资源专业知识以及对公司文化、工作场所动态和福利的见解。

布伦南是顾问公司的首席顾问。她是广受欢迎的全球演说家和培训师。布伦南专注于创新而可行的人才获取、员工参与以及与业务成果相关的未来工作策略。

[](https://medium.com/datadriveninvestor/corporate-cultures-future-is-now-2f7fd6bfcaa1) [## 企业文化的未来就在现在

### 企业家需要认真审视他们今天的处境

medium.com](https://medium.com/datadriveninvestor/corporate-cultures-future-is-now-2f7fd6bfcaa1) 

两人都比较了自己的公司在工作场所文化方面与竞争对手的不同之处。

布伦南说:“确保你的差异化从你的招聘启事和求职网站开始。”。“如果你可以替换你的名字，并使用你的竞争对手，这不是差异化。

“你如何展示这些东西来区分？”她说。“任何公司都会犯的最大错误之一就是认为自己没有竞争对手——尤其是在招聘方面。如果你认为自己没有，那你就没有找对地方。”

# 出示你的身份证

公司的身份是一个关键的区别。

“竞争有很多层次，”布伦南说。“我在制造业和非营利组织遇到的许多客户都很难接受这个概念。他们在发展身份认同时会挣扎。”

G&A 合伙人让他们的工人参与到这个过程中。

“我们有一个每周 eNPS——员工净推介值——计划，”马丁说。“它要求我们的员工对 G&A 进行评级，并提供反馈，说明哪些方面有效，哪些方面有待改进。

“这使得领导层能够立即补救可能影响员工士气或公司文化的问题，”她说。

[](https://medium.com/datadriveninvestor/engagement-is-a-four-letter-word-e49970cdd3fb) [## 订婚是一个四个字母的单词

### 工作场所文化弥合了沟通上的差距

medium.com](https://medium.com/datadriveninvestor/engagement-is-a-four-letter-word-e49970cdd3fb) 

招聘策略因公司而异，成功程度也各不相同。

“我们专注于战略采购策略，寻找那些拥有知识、技能和能力的被动候选人，以有效地完成我们招聘的角色，”马丁说。“我们还鼓励员工推荐他们认识并愿意合作的人来填补空缺。”

# 经验教训

布伦南深入研究了公司如何从错误中学习。

"什么东西在你以前工作的地方有效，但在你现在的公司无效？"她说。“当文化发生变化时，并非所有伟大的战略都同样有效。

“你可以有一种文化，但不能定义或控制它，”布伦南说。

她说，这些要素在招聘中很管用:

*   老实说在这个过程中。
*   快速反应。
*   主动战略或采购。
*   有意义的采访。
*   尊重候选人。
*   不依赖一个来源。
*   推荐。

布伦南说:“对于无法对所有员工都这样做的公司，我的一位客户为全包式度假制定了一个计划。”。“这花了他们 3000 美元。他们增加了 125%的推荐。”

[](https://medium.com/datadriveninvestor/special-benefits-are-the-decider-for-workers-happiness-bdc4b2207410) [## 特殊福利是工人幸福的决定因素

### 正确的程序阻止人们走出大门

medium.com](https://medium.com/datadriveninvestor/special-benefits-are-the-decider-for-workers-happiness-bdc4b2207410) 

马丁描述了 G&A 的求职过程。

“一旦我们确定了一个可行的申请人，我们接下来的步骤包括最初的电话筛选、评估、与招聘经理的面对面面试，并在做出招聘决定之前让候选人保持知情，”她说。

马丁说:“不太适合这个职位的申请人会被审查，并考虑其他职位。”。“另一种选择是，我们给他们发一封通知，告诉他们我们目前不会继续聘用他们，但会将他们的简历存档，以备将来招聘之用。”

最理想的情况是，这不会是一个永久的中断。

马丁说:“任何公司对招聘流程所能做出的最有影响力的改变，就是找到一种方法，让你系统中已经存在的潜在未来候选人参与进来。”

布伦南有一个问题要问一般的人力资源主管。

"你最后一次在自己的求职网站上申请工作是什么时候？"她说。“我向你挑战这么做。

布伦南说:“作为一名新申请者，找到并申请。”“你们中的许多人会对所花的时间感到惊讶。问题通常是公司的决定，而不是你的求职者跟踪系统。”

# 入职试验

许多公司在新员工入职后不久就招聘新的团队成员时犯了错误。

布伦南说:“入职与保留、客户满意率和盈利能力直接相关。”“你的高管团队愿意放弃哪一个没有做好的？”

马丁表示同意，并补充了另一个要素。

“你必须为新员工提供适当的入职培训，”她说。“如果新员工一开始就不觉得他们已经为成功做好了准备，以后就很难说服他们。”

[](https://medium.com/datadriveninvestor/cut-excuses-for-bad-onboarding-experience-498913851acb) [## 为糟糕的入职体验找借口

### 对公司的良好介绍有助于幸福的婚姻

medium.com](https://medium.com/datadriveninvestor/cut-excuses-for-bad-onboarding-experience-498913851acb) 

职位发布的方式对申请人的最初影响最大。

“用简短的短语开始招聘，”马丁说。“提供公司的简要亮点，以便尽早吸引求职者的注意力。让招聘信息简短易读。然后候选人可以很快决定他们是否有资格或对这个职位感兴趣。”

布伦南建议确保公司得到“正确的”候选人:

*   不要贴职位描述。招聘启事是一种营销工具。
*   不要使用所有蓬松的语言。谈论这份工作和它的作用，它的价值等等。
*   检查[分集偏差](https://medium.com/@JKatzaman/diversity-quest-is-a-must-and-never-ending-901eb9a73257)。

马丁和布伦南权衡了员工推荐流程。与传统应用程序相比，它能产生更好的候选人吗？

马丁说:“如果做得好，员工推荐过程也会同样有效。”“为被聘用并停留超过 60 至 90 天的推荐人提供奖金是激励员工提供更多合格申请人的一个好方法。”

布伦南也支持员工推荐。

“有大量的研究表明这是事实，”她说。“如果你的公司认为这不可行，那么你拥有的是一种文化和参与度，而不是招聘问题。你将永远无法从糟糕的文化中找到出路。

# 不要责怪推荐人

“推荐不需要雇佣，”布伦南说。“糟糕的推荐招聘不是推荐他们的员工的错。这是人力资源和雇佣他们的经理们的错。”

一家公司的品牌是潜在雇主的简历，供候选人权衡申请是否值得。企业和工人互相雇佣。

[](https://medium.com/datadriveninvestor/human-touch-boosts-outstanding-brands-to-the-top-5b71cd610e73) [## 人情味助推优秀品牌走向巅峰

### 消费者的购买行为与情绪息息相关

medium.com](https://medium.com/datadriveninvestor/human-touch-boosts-outstanding-brands-to-the-top-5b71cd610e73) 

马丁说:“雇主品牌不仅仅是为了让公司的名字家喻户晓而进行的营销。”。“你的品牌元素——使命、价值观、文化——响亮而清晰地表明了这家公司是什么样的，以及潜在候选人如果在你的公司工作会有什么样的期望。

“一个积极的雇主品牌可以让招聘过程变得容易得多，”她说。“顶尖人才更有可能对你做出回应，甚至主动寻找你的公司。”

公司创造他们的品牌，或者说品牌是为他们而创造的。

布伦南说:“如果你认为自己没有就业品牌，那你就有。”“你不是控制和定义它的人。别人是。不要那样做。

她说:“一家雇佣品牌或形象不佳的公司比一家正面或中立的公司平均多支付 10%的薪水和福利。”“雇佣品牌影响底线结果。”

招聘人员在更好地将公司定位于理想候选人方面发挥着核心作用。

“他们必须了解公司努力实现的目标，以及什么样的人可以帮助他们实现这些目标，”马丁说。

她说:“从那里开始，一切都是为了找到符合条件的顶尖人才。”。“向他们提供一个价值主张，清楚地定义如果他们加入公司，他们会得到什么。”
