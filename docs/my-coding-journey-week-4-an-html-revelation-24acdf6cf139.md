上周，我记录了我使用稍微更有野心的 CSS 动画的经历。虽然我取得了一定程度的成功(在某种程度上，炸掉卡通兔子的头可以被认为是成功的)，但和往常一样，我也有相当多的沮丧，与响应有关。我想我知道*我需要做什么*来解决这个问题(相对于预爆炸兔子头的尺寸来设置段落尺寸)，但是我还不知道*如何*去做。总的来说，这有点像荷马在组装烤肉。

它向我展示了如何:
