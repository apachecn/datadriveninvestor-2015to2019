Kubernetes 是 Google 的开源容器编排框架。事实上，它们是一堆 Docker 微服务 Kubernetes。

Kubernetes 允许您在几秒钟内安装和部署您的应用程序，并且它具有健康检查的功能。如果应用程序不能正常工作，我们就终止它，在另一台机器上重启它。如果集群中的一个节点出现故障，我们会将应用程序转移到另一个节点。这都是自动完成的。

**为什么 Kubernetes 比其他管弦乐队优秀？**

Kubernetes 是最受考验的，它不属于任何公司。好像后台有很多玩家有机会发展它。如果一家公司破产，我们将一无所有。Kubernetes 现在是容器编排事实上的标准。它被思科、IBM、谷歌、英特尔、微软使用。一般来说，所有的大公司都已经在生产中使用它了。

除了 orchestrator 为开发人员提供工作这一事实之外，这对企业来说也是一个很好的经济效益。它可以让您大大节省技术和人力资源。让我们来看看使用 Kubernetes 可以获得哪些好处。

**盈利 1 稳定**

第一个优势是，你不再有这些棍子和透明胶带围绕着你的装置，也就是说，一切都像主流。它花费很多。

**利润 2 减去涉及的技术数量**

它允许您尽可能多地处理机器。一般来说，如果没有 Kubernetes，您将启动一个应用程序，启动第二个虚拟机，启动第二个应用程序。Kubernetes 以这种方式自动转移和放置节点，放置应用程序以最大限度地利用处理器负载。
