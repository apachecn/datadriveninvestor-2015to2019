# 不要假装人工智能会解放员工的时间

> 原文：<https://medium.datadriveninvestor.com/stop-pretending-ai-will-free-up-employees-time-99cb4eb4608f?source=collection_archive---------3----------------------->

最近，我参加了一个关于人工智能未来的小组，一名小组成员面无表情地声称，他认为他的公司新的会计人工智能解决方案将对会计师有益。因为，正如他解释的那样，通过自动化 30%的工作，这意味着会计师可以花更少的时间在工作上，花更多的时间在家庭上。

我几乎想嘲笑我认为是超级无聊的幽默，直到我意识到他是认真的。所以，我问，“你真的认为公司投资数百万在自动化上是因为他们希望他们现有的全体员工能够早点回家吗？你相信这些痴迷于削减成本的企业集团的驱动力吗？确保员工最终能够在午餐后不久就收工回家？”

### 随着技术颠覆和自动化与日俱增，关于……问题的争论也越来越激烈

www.datadriveninvestor.com](https://www.datadriveninvestor.com/2018/10/23/technological-disruption-and-the-future-of-jobs/) 

这位小组成员承认，肯定会有一些工作岗位流失，但他不想显得愚蠢天真。对此我进行了澄清，确切地说，30%的工作岗位将会流失。

可悲的是，这种天真的情绪并不仅限于乌托邦式的初创企业创始人。最近，在展示他们新的店内机器人时，沃尔玛声称他们想解放员工，让他们有更多的时间与顾客在一起。

这就引出了一个问题，有没有人真的觉得他们目前没有足够的时间与沃尔玛的看门人见面？

这有点技术精英主义的味道，假设工人阶级如此容易受骗，以至于看不到真正发生了什么。这就像试图说服某人拆除他们的房子与新的商业开发无关，但你只是担心他们没有得到足够的新鲜空气。

最终，这只是一种干扰，让我们无法进行我们现在应该进行的更困难但必要的对话。像这样的问题，我们如何确保自由职业经济中工人的稳定性？注重遵守规则教育的学校是否为学生做好了毕业时已经自动消失的工作准备？或者，我们如何确保自动化工作的好处分配给所有人，而不仅仅是技术的所有者？

但我认为，面对社会未来的生存问题是一项艰巨的任务。因此，更容易把这一切抛到一边，转而从事异想天开的幻想，其中唯一棘手的问题是，我们多早把长期工作不足的会计师送回家，以便他们可能有更多时间与前看门人变成私人购物助理一起仔细查看当地沃尔玛的过道。
