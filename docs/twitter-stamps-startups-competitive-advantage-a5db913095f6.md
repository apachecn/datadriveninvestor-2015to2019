## 聆听竞争对手的推文，并与他们的观众互动

![](img/9cd8f889ff0687247d9b481c3dc19c67.png)

对于企业家来说，社交媒体可能不是最重要的。然而，它有巨大的投资回报潜力。

[莫迪凯·霍尔茨](https://twitter.com/mordecaiholtz)是这方面的专家。Blue Thread Marketing 的联合创始人兼首席数字媒体策略师，他与八个国家的各种客户合作，从初创企业到耶路撒冷市。

[](https://www.datadriveninvestor.com/2019/02/26/social-media-listening-keeps-you-plugged-in/) [## 社交媒体倾听让你时刻保持联系——数据驱动的投资者

### 虽然社交媒体倾听是一种顶级趋势，但它应该是每个现代营销策略的一部分。保持和谐…

www.datadriveninvestor.com](https://www.datadriveninvestor.com/2019/02/26/social-media-listening-keeps-you-plugged-in/) 

“我开发独特的内容策略，与每个品牌的叙事产生共鸣，”他说。

霍尔茨与社交媒体营销企业家和 Twitter 爱好者 [Madalyn Sklar](https://twitter.com/MadalynSklar) 谈论了行业趋势、搜索选项、黑客和其他有用的 Twitter 技巧。

如果营销不善，即使是伟大的产品也会失败。与这种前景相反，Twitter 给了初创公司竞争优势。

“Twitter 允许一家初创公司分析竞争对手的推文；学习、互动和模仿他们的观众；并了解行业趋势和内容。它是免费的。它很简单，可搜索。是实时的。这是一个极好的内容来源。它打破了最初的网络障碍。

“设置搜索你的竞争对手的处理，”他说。“了解人们在说什么。访问他们在其他社交网络上的活动。”

# 打破僵局

初创公司寻找特定的受众，尤其是如果他们在 Twitter 上。

“社交是最终的破冰船，”霍尔茨说。“初创公司希望并需要消费者基础、思想领袖网络和接触潜在投资者的渠道——所有这些都在 Twitter 上。

“许多投资者和思想领袖会在互动前检查初创公司的社会证明，”他说。如果竞争对手基于内容获得了关注，开始使用它。如果不是，那就是即时优势。"

[](https://medium.com/datadriveninvestor/surviving-fundraisings-slings-and-arrows-a0158c6857cc) [## 幸存下来的筹款的吊索和箭

### 筹集资金时，找你可以信任的有经验的人

Twitter 还可以用来监控行业趋势，这对初创公司很有用。

霍尔茨说:“建立监控谈话的清单。”“这样，创业公司可以了解情绪，锁定他们的受众，并了解压力点。

他说:“将趋势与最新的热门话题同步会将活动推向新的高度。”“精心策划，让初创公司以一种有助于学习、成长和实现目标的方式互动。”

# 寻找和发现

创业公司也需要知道 Twitter 的搜索选项。

“创业营销是一门科学，”霍尔茨说。“初创公司可以进行竞争分析，在行业内寻找话题，并与全球目标受众进行实时对话。”

Twitter 上有一些针对希望跟上行业领导者的想法和思想的创业公司的方法。

[](https://medium.com/datadriveninvestor/startups-and-investors-seek-the-right-fit-57671d9c07d6) [## 创业公司和投资者寻求合适的契合点

### 企业家必须问自己是否做好了投资准备

medium.com](https://medium.com/datadriveninvestor/startups-and-investors-seek-the-right-fit-57671d9c07d6) 

“创建列表来汇总他们的内容，”霍尔茨说。“制定行动计划，如销售单。要深思熟虑。主动互动加自动化。

“用不同的标题和图片调整和优化内容，”他说。“利用周末作为接触和与人交流的最佳时间。让他们在内容上成为思想领袖。”

如果适度的话，转发对创业公司也有重要的作用。

“从目标人群列表中分享内容，并从你的启动页面上分享，”霍尔茨说。“不要过度。”

# 艰难的接触

他举了几个例子来说明 Twitter 是如何被用来联系那些无法联系到的人的:

*   创建顶层人员列表。管理内容，定期与列表互动，鼓励他们参与。
*   在传统营销活动中加入标签。使用标签和逆向工程内容策略，如关键字。

“长话短说，如果你说了一些引人注目的事情，你可以和首席执行官、名人以及你读的上一本书的作者交谈，”霍尔茨说。"神奇之处在于互动的简单性."

当与有影响力的人交流时，不要问他们已经被问过一百万次的问题。做一点调查。表现出一些主动性。

“就像一个老式的冷电话，”霍尔茨说。"在你拿起电话或发微博之前，了解你的听众."

Sklar 证明了这种方法的成功。

“我一直在做这件事，”她说。“当我读到这些大人物的书或在播客上听到他们时，我会给他们发推特。我在推特上大声呼喊。这一直都管用。”
