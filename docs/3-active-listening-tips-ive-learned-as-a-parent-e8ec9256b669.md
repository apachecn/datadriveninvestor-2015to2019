上周就是如此。我(反复)学到的教训是，不要试图解决或解决问题，而是要充分倾听。倾听别人在说什么。

虽然这听起来很简单，但事实并非如此。我们中的许多人都很难主动倾听，因为我们习惯了被动倾听(等待一会儿说话，而不是真正听到对方在说什么)。

我的知识来自于我作为母亲的角色。我的女儿在学校与女孩戏剧斗争(你父母知道我的意思)。这件事彻底震撼了她，深深地伤害了她。

当她(含着眼泪)告诉我她的故事时，很明显她觉得没有人在听她说话。没人听是她苦恼的根本原因。所以我把工作和晚饭放在一边，花时间积极地听她说话。

以下是我花时间(是的，不止一天)微调积极倾听技巧的一些顿悟时刻。

# 1.习惯于沉默

当一个外向的人听一个内向的人讲述她的故事时，我学到的第一件事就是不要用言语来填补沉默。沉默并不是邀请我说话。相反，寂静是她收集思想、处理信息和自我反省所需要的空间。这些是她沟通风格的关键要素。在这种沉默中说话会打断对话，而不是推进对话。

# 2.超越言语

超过 80%的交流是非语言的，我女儿的肢体语言告诉我的比她说的话更多。她一边说着，一边摆出一副乌龟缩进壳里的姿势。她的背弓着，她的胳膊紧紧抱着她的腿靠近她的胸部。

虽然这是一个明显的肢体语言提示，但它提醒我们，不仅要听这个人在说什么，还要听他们的肢体语言是否与这些词语一致。他们有开放的肢体语言(手臂放在身体两侧，心连心地面对你，进行眼神交流)还是封闭的肢体语言(双臂交叉，看向别处，拨弄头发或衣服)？注意这些暗示会让你听出正在说的话，更重要的是，听出没有说的话。

# 3.这不是关于你的

积极倾听是指充分倾听对方。就这么简单。

要成为一个积极的倾听者，你需要在场。在我和女儿的对话中，我没有给她建议，也没有告诉她不要再敏感。这些评论对她没有帮助(因为她的肢体语言在说，她无论如何也不会听到我的话)。

相反，我反复告诉她——我听到了。每次看着她的眼睛。

随着时间的推移，她对冲突的压力和焦虑减轻了，她打开了谈论冲突的空间，听我讲述了童年时被欺负的故事。我们还看了一些关于移情的视频，帮助她理解她并不是太敏感，而是她接收信息的方式。

# 熟能生巧

在接下来的一周里，我邀请你在你的职业或个人生活中练习一些积极的倾听技巧。当你意识到自己在寻找插入对话的方法时，深呼吸。问问你自己——你是真的在听别人说话还是只是在等着说话？

使用积极的倾听技巧如何改变谈话以及你的人际关系？记住，通过舒适的沉默，我们给了对方空间来完成他们的想法或反思他们所说的话。
