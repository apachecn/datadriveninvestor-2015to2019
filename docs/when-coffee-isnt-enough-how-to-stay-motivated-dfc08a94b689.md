# 当咖啡不够时——如何保持动力

> 原文：<https://medium.datadriveninvestor.com/when-coffee-isnt-enough-how-to-stay-motivated-dfc08a94b689?source=collection_archive---------9----------------------->

[![](img/8be258916ae36f9cdd949881589e28b0.png)](http://www.track.datadriveninvestor.com/1B9E)![](img/9e7a7a00dbb2b22a0dbb4d42318a5545.png)

你是否曾经在一个项目中工作过，或者在一份工作中，你几乎没有动力去完成它？你曾经拥有的火花早已消逝，但你仍日复一日地继续努力。

我们都经历过。很多次。

那么如何找到自己走出窠臼的动力呢？没有神奇的公式，因为激励我们每个人的是不同的。

虽然有些人受到金钱的激励，但大多数人受到金钱买不到的东西的激励。

*   与同事和/或领导的积极关系
*   尊重和支持的工作场所文化
*   学习和发展的机会
*   有意义的(并得到支持的)工作生活平衡

这里有一些建议可以帮助你找到保持动力的方法。

# 第一步——评估

从评估什么激励你开始，回顾上面的列表。这些要点是否反映了您当前的工作环境？如果有，有多少？如果没有，记下你理想的工作环境是什么样的。

# 第二步——找到你的动力

弄清楚什么会激励你。回想一下你曾经兴奋的一份工作或一个项目。是什么激发了你的兴趣？你早上来上班的时候是什么感觉？当你在做自己喜欢的事情时，你的表现是怎样的？

# 第三步——需要改变什么

看看你目前的情况，有什么因素会激励你吗？或者是不是已经到了组织或工作不再适合你这个人或专业人士的时候了？没事的。随着我们的成长和发展，总有一天我们会超越目前的工作、友谊，甚至是我们对职业道路的期望。

# 第四步——拥有改变

对许多人来说，这是最难的部分。承担变革的责任。只有你知道如何保持动力。没有人能为你做这件事。也许你需要参加一个课程(提高或完善你的技能)。这可能意味着改变组织内部的部门或角色(如果你喜欢企业文化，但不喜欢你的直属工作团队)。也可能是在刷简历，找新工作。无论需要什么样的改变，都由你来做。只有你。

# 第五步——持续登记

不要只做一次这个练习，不要因为一份稳定的薪水支票或养老金计划而沾沾自喜。相反，继续和自己进行诚实的对话。你所做的改变是否提高了你的动力？多少钱？你被你所做的工作激励了吗？如果是的话，恭喜你。但是如果你仍然不完全满意(人生苦短，不能过苦日子)，那就深入挖掘。

我鼓励你每年至少做两次这样的检查(不妨现在就开始)。否则，你可能会发现很多年过去了，有一天你醒来时会想为什么你不快乐，你是怎么来到这个地方的。

通过了解激励你的是什么，你可以确保它成为你日常生活的一部分——无论是个人生活还是职业生活。
