# 人工智能如何帮助解决最后一公里的配送问题

根据美国人口普查局的数据，已经占零售总额 9%的电子商务正在以两位数的速度增长。什么也在快速增长？人工智能在商业中的应用。以下是人工智能如何帮助解决电子商务的最后一英里配送问题。

普华永道的战略咨询业务 Strategy&估计，2018 年将交付 110 亿个包裹，到 2020 年这一数字将跃升至 160 亿个。顾问进一步预计，电子商务将占这一数量的三分之二。毕竟，电子商务提供了:

*   更广泛的选择
*   更简单的价格比较
*   更方便

然而，零售商面临的一个挑战是，消费者也希望他们订购的商品能更快地送达。最后一英里配送问题是，零售商需要有效地将产品分发给全球消费者，通常依赖外部合作伙伴(联邦快递、UPS、随机白色货车中的夫妻店配送服务)将包裹送到消费者手中。不是每个人都有预算或技术来尝试送货无人机或机器人！

# **人工智能如何帮助**

亚马逊显然是这一领域的巨头，它很快将拥有自己的本地货车车队(由独立公司运营)。这使得其他企业在依赖运输系统的同时寻找竞争方式，而该运输系统并不能满足快速发展的电子商务的需求。障碍包括:

*   当前系统的成本效率侧重于长途旅行
*   无法应对波动的需求
*   缺乏本地送货服务
*   国家和地方运输提供商之间几乎没有重叠
*   难以在较小范围内协调努力以保持盈利。

> 正如 Strategy& consultants 观察到的，“盈利能力仍然高度依赖于两个关键因素:(1)路线密度——在给定的递送运行中可以递送多少包裹，以及(2)投递量——在每个站点递送多少包裹或物品。”

与此同时，客户不愿意支付太多的运费，即使是当天送达。根据小型企业趋势，如果运输价格过高，28%的网上购物者会放弃他们的购物车。

然而，让人工智能来协调和管理最后一英里的交付可以产生积极的影响。随着人工智能进行强大的计算和机器学习，分析和观察历史趋势和预测模式，选项可以与人工智能进行快速成本效益分析，以确定如何交付包裹。

人工智能可以帮助更有效地运送货物，这是盈利的关键。除了简化流程，人工智能还可以帮助电子零售商及其运输合作伙伴根据历史数据更好地预测出货量，这可以帮助送货服务提前规划，以度过黑色星期五运输期和 2 月中旬死区之间的波动。

随着人工智能影响最后一英里的交付问题，区块链也可能发挥作用。例如，它可以为交易所提供主干网，让交易所实时看到最后一英里递送服务对包裹的竞价。区块链还可以提供零售商和运输公司在后端寻求的安全性。

研究表明，最后一英里可能是三分之一运输成本的根源。随着人工智能驱动的交付和路线管理算法推动物流创新、简化流程和提高效率，最后一英里交付问题可能会变得不那么令人生畏。
