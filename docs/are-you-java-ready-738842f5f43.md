# 计算机概念

开始使用 Java 或任何编程语言时，了解一些计算机基础知识会有所帮助。定义这三个术语可以帮助你进入软件开发。

# 投入

我们可以从许多事物中获得输入。例如，我正在输入的键盘是一种输入形式。我们使用的鼠标或者你手机上的手指是输入。

# 输出

每当你查看屏幕或阅读报告时，你都在消耗输出。输出并不总是以人们可读的格式。来自计算机的许多类型的输出被其他计算机或系统消费。

# 数据

我们生活在信息时代。Alexa 可以通过你的声音找到数据。所以你想知道巨人队的比赛比分？比赛什么时候开始？数据存在于我们无处不在的计算机的许多地方。

# 编程概念

我妻子在金融服务业工作。他们有听起来像另一种语言的行话。我已经学得足够精通了。编程也是类似的努力。我们用概念和术语来描述我们正在做的事情。让我们介绍一些基础知识来帮助您开始。

# 句法

所有语言都有自己使用的语法。Java 有表示特定事物的关键字。例如，有一个“For”循环。它将检查列表中每一项的代码片段。编译器很像你八年级时遇到的那个严厉的英语老师。对于错别字是毫不留情的。

# 算法

这个词听起来比实际上更有气势。算法就是我们完成一个过程的简单步骤。在银行业务中，借记一个账户是有步骤的。我们希望谨慎行事，这样就不会有人透支。那会让一些人生气的！

# 应用

为了追求简洁，我们将应用程序缩短为应用程序。我们到处都有。在我们的手机、手表和电视上。我们最近转到了 Youtube 电视。在我们的 [LG 智能电视](https://www.lg.com/us/tvs)上很容易。我们刚刚安装了一个应用程序。应用程序只是在设备上运行的代码的集合。
