个性化医疗保健是不可避免的，因为我们生活在一个数据技术让每个人都可以访问各种信息的时代。与此同时，消费者的期望值越来越高，预期寿命也越来越长。

基于价值的医疗保健是一种交付模式，其中包括医院和医生在内的提供者根据患者的健康结果获得报酬。根据基于价值的护理协议，提供者将因以循证方式帮助患者改善健康、减少慢性病的影响和发病率以及过上更健康的生活而获得奖励。

得益于互联网，20 世纪 90 年代是信息技术革命的十年。现在我们生活在根据这些信息采取行动的时代。

这就是个性化和个人选择进入医疗保健的原因。人口健康是一个良好的开端，但还不够。我们不能遗漏整个患者群体，例如少数民族。

人工智能和机器学习是医疗保健有效个性化的基础，因为:

*   个性化要求精确和快速
*   它降低了成本
*   减轻医生和其他提供者的压力
*   它有助于无歧视地为大量患者提供护理

医疗保健的价值是对医生向患者提供的医疗服务质量的衡量，仅基于他们之间独特的相互关系，并考虑到[患者生活的各个方面](•%09https:/futurism.com/ai-creates-complete-3d-models-cells-letting-us-see-diseases/)，包括社会、情感、身体、财务、环境、家庭、基因、精神、意识形态和个人。

**医疗保健的价值是什么？**

医疗保健的价值是基于政客和其他不从事医疗保健的第三方创造的有偏见的标准吗？价值只是一个吸引注意力的时髦词，而没有传递任何实际价值吗？

医疗质量不是一个数学公式。它不能通过命令、政策甚至新的商业模式来强制执行。在以人口为基础的医学上分配以价值为基础的措施是不合适的，也是失败的。企业和其他第三方设定的价值没有考虑到人的因素。

质量是指在给定可用选项的情况下，与患者和医生的期望和能力相匹配的医疗服务中的社会认可和最大利益。

有许多公式和方程来评估护理质量，但底线归结为变量。这些变量是由地理、文化、遗传、环境、教育、疾病以及最重要的个人需求决定的。

**个性化医疗意味着什么？**

希波克拉底说，“有时治愈，经常治疗，总是安慰。”但是每个人都有不同的需求。

今天，当我们谈论个性化医疗时，大多数人会想到基因组学，但个性化并不以基因组学或基因干预为起点和终点。它要宽泛得多。

任何技术和策略都被认为是个性化医疗，这种技术和策略可以定制一种治疗方案，以最小的副作用和并发症为患者提供精确的治疗。

如今，每个人都在谈论“在基于人群的最佳价值补偿模式背景下，以患者为中心的医疗保健”。这是不对的。价值和报销是风马牛不相及的。价值是指为个人量身定制的护理质量，而不是保险公司。

希波克拉底的医疗保健理念是个性化医疗保健。他不会赞成由政客和保险公司主导的一刀切模式。
