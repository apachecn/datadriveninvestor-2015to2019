# 大声沉默:什么都不做怎么能回报最多

> 原文：<https://medium.datadriveninvestor.com/loud-silence-how-doing-nothing-can-return-the-most-543d7902b626?source=collection_archive---------30----------------------->

[![](img/0654f3b19b7ee612c0bb3ee8c0ca0e35.png)](http://www.track.datadriveninvestor.com/1B9E)![](img/79eefa49e7ef8fc9c88baf729f025cde.png)

Hendrik Bessembinder 教授今年早些时候进行的一项研究强调了股票市场净收益的极端差异:“当以一生的美元财富创造来说，表现最好的 4%的上市公司解释了自 1926 年以来整个美国股票市场的净收益，因为其他股票集体匹配国库券。”所以，Bessembinder 先生说的是，在过去的 92 年里，回报率最高的 4%的上市公司占了整个 T4 股市的全部净收益。雪上加霜的是，市场上其他 96%的公司表现类似于美国国债。对于那些不熟悉的人来说，这些美国国债被广泛誉为最安全的投资(风险最小也意味着回报最小)，年回报率仅为 4%左右(在最好的情况下)。

谈论生活在极端主义…

这项研究最令我惊讶的是，这不仅仅是过去 10-15 年的一个片段，我(我敢肯定，还有许多其他人)认为这是最不稳定、最有利于收入不平等的时期，但这种财富的高度集中塑造了过去一个世纪的市场。不要妄下毫无根据的结论，但如果这种情况已经持续了 100 年，那么有理由假设这种情况还会持续 5 年、10 年、20 年。

我提到这一点是因为如今，投资世界(以及整个世界)正变得更加自动化，竞争更加激烈，信息更加泛滥。人们当然很容易被过剩的数据和潜在的机会吓倒，而看上去“最忙”的投资组合经理和投资银行家给人的感觉是，他们是唯一能够真正掌握所有正在发生的事情的人。但是，就像股票数据一样，我正在训练我的神经网络，这些数据大多充满了分散注意力的、无意义的噪音。仅仅因为你的投资组合尽可能多样化，或者拥有数百只来自不同领域的不同股票，或者你每天都在改变你的投资组合，这并不意味着你有更好的投资策略。你可以投资 1000 只股票，这些股票都属于收入最低的 96%，这样你就能收支平衡，而你的同事只持有一只属于那 4%的股票，并在接下来的几年里大赚一笔。

沃伦·巴菲特(Warren Buffet)的一句话真正总结了这一点，他说:“我可以通过给你一张只有 20 个槽的票来改善你的最终财务福利，这样你就有 20 次机会——代表你一生中要做的所有投资。一旦你在卡上打卡，你就再也不能进行任何投资了。”

有时候，最好保持警惕，不参与其中，直到你发现一个大的投资机会。我把这想象成在渔船上的一天:你可以一次使用所有的鱼饵，只是为了放线而放线，或者你可以坐在那里，直到一条大鱼上钩。

现在我想到了我正在构建的交易算法(当然不止我一个！)，这绝对是一个值得牢记的有趣话题。Word2Vec 模式映射不可能始终为所有实时数据在股市中找到类似的价格变化。如果我的代码可以全天候匹配类似的股票事件，那么这意味着我的代码被破坏了。在我的例子中，最好是一直运行事件匹配代码，但是只在事件匹配超过某个准确度阈值时才考虑投资(现在，让我们称之为准确度阈值)。如果两个事件以 100%的相似性匹配(这是极不可能的，我只是用这个作为例子)，那么它意味着一些重复模式将重复的真实可能性。一旦代码越过了准确性阈值，我们就需要看看在我们匹配的 data_embedding 之后发生了什么。换句话说，如果我们的实时数据紧密映射到一个 data_embedding，而 data_embedding 之后价格上涨了 12%，那么它绝对值得投资。问题是将百分比相似性(应该转换为我们的实时数据将遵循它所映射的数据的相同结果的可能性百分比)与预期收益考虑在内。具有高预期增益的非常高的相似性百分比是最好的情况，而具有高预期增益的低相似性百分比更不明确。

出现的问题是，如果我们的实时数据与我们匹配的历史事件后发生的情况类似，我们如何将历史股票市场事件和实时股票市场数据之间的匹配百分比计算在内？

因为毕竟，我们需要能够在大鱼上钩时识别它们。
