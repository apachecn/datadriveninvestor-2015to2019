进入**文件**选项卡，点击**选项**

From File click on Options

然后点击**插件**类别。

![](img/9dc85044ac3bf66520735524720a4127.png)

Click on Add-ins

在**管理**框中，选择 **Excel 插件**，然后点击**转到**

![](img/74a90e800dc3a255e57d5f214c76cf07.png)

Manage Excel Add-ins

在**可用附加模块**框中，选择**分析工具箱**复选框，然后点击**确定**

![](img/b64811aa20b6a1f857aa077ca263a2dd.png)

Analysis Tool Pack

你现在应该可以在数据选项卡下看到**数据分析**命令。
