你不需要学着成为你想做的每件事的专家。事实上，你可以通过自己的实验和实践学到更多。这就是数据科学的美妙之处。你可以做任何事。

以下是学习数据科学的几种主要方法:

# 1.和朋友一起建立一个数据科学团队。

没有人，你什么也学不到，这是最好的方法之一。带上你的朋友，参加黑客马拉松。或者，如果您不认识数据科学领域的任何人，您可以自己动手，参加不同的数据科学活动来结识他人！

# 2.看了很多数据科学的书

# 3.开始写博客

许多数据科学家撰写文章和博客，讲述他们的经历，这些经历可以教会你很多关于数据科学世界实际上是如何工作的。这是一个很好的学习方法。

# 4.在网上上一些课程。

有很多在线课程，甚至有些是免费的。我最喜欢的是 Coursera 系列，它使用在线形式教授数据科学。Coursera 有来自一些顶级数据科学教授和学生的数千门课程。

# 5.在线研究数据科学工作

许多顶尖的数据科学公司都有发布在招聘网站上的职位。你可以找到实习机会，看到更高级职位的要求，这样你就知道下一步该学什么了！

# 6.观看一些数据科学家的演讲

数据科学讲座在许多会议中非常受欢迎。看看这些演讲，看看你还能学到什么有趣的东西。

# 7.阅读大量关于数据科学的文章和博客

阅读您将在工作中看到的有趣的概念和工具。然后，回到这一页，看看你已经读过的东西，看看你是否还有问题。
