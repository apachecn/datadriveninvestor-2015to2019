然而，随着许多研究人员可以对 RNA、内含子、外显子、基因表达、微小核糖核酸等进行测序，甚至基因标记也开始变得模糊**。随着技术变得更容易访问和使用，数据财富将继续增长。**

**但这也意味着，随着我们向混合体中添加更多的数据，描述对人类意味着什么的信息的复杂性也在增加。**

**现在，正如我们所知，成像领域经历了巨大的飞跃和进步，随后在不同领域创造了大量价值。如果没有深度学习技术，像自动驾驶汽车和肿瘤分割这样的应用是不可能的，这些技术现在正充斥着世界各地的 Github 帐户。**

**但是，如果我们把这两个领域结合起来会怎么样呢？这似乎是朝着结合多模态健康数据的方向迈出的自然一步。**

**这就是数据和人工智能的美妙之处。它让我们可以利用大量的技术来寻找以前不太明显的有趣变量之间的相关性。**

**首先，影像遗传学的一些应用包括但不限于:**

1.  **一种通过遗传易感性了解神经退行性疾病(如帕金森氏症和 ALS)中大脑变化的方法**
2.  **一种创建遗传风险评分的方法，该评分由某些单核苷酸多态性的存在与否组成。**
3.  **一种揭示细胞生长或死亡与特定人群遗传之间未知关系的方法。**

**世界是我们的牡蛎，我相信这些多模式方法的进步将成为真正理解这些多方面疾病的答案的一部分，这些疾病对于科学家和临床医生来说很难掌握。**

**当我们通过新兴技术使彻底调查成为可能来创造更多的复杂性时，我们就有希望理解得更多。**

**有一点是肯定的——所需的人力和智力资本将会相应增长。**
