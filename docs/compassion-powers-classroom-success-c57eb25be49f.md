# 同情心是课堂成功的动力

> 原文：<https://medium.datadriveninvestor.com/compassion-powers-classroom-success-c57eb25be49f?source=collection_archive---------2----------------------->

![](img/096c73565f8588077df7939379f91595.png)

与学生建立情感联系始于同情。你印象深刻的老师最有可能带着同情心来教学，而不是要求死记硬背和背诵。

[约翰·米勒](https://twitter.com/mexusmx)，斯蒂芬妮·温德尔和[詹姆斯·瓦拉克](https://twitter.com/jamesvarlack)谈论了同情心以及它如何给他们的课堂带来生活和成功。

同情是对他人有感觉，在需要时提供支持——不等待被要求就去要求。

“我对同情的定义是，当你的心与别人的需求发生冲突，你有责任采取行动，”三年级教师温德尔说。

米勒是墨西哥城一所国际美国学校的负责人。

“同情是承认他人的斗争，并以真实的方式与这种斗争联系起来，”他说。

“在教育和技术之间架起桥梁”的教育家 Varlack 希望释放被压抑的同情心。

# **内部对外部** l

“当我们从内部表达转移到外部表达时，会有更多的力量，”他说。“不仅要看到需求，还要采取行动。

“我们说了很多关于同情心的需要，却从来没有付诸行动，”Varlack 说。“我们需要拥有它。”

每个人都回忆起作为一名教育者经历过同情。

米勒说:“当我儿子生病时，我的同事们竭尽全力帮助他。“我们成了一辈子的朋友。”

温德尔自己也面临着艰难的考验。

“今年我有一个新学生，他对我来说是个棘手的家伙，”她说。“我很抱歉地说，我对他的看法和反应只是在我得知他的父母之一被监禁后才改变的。

“他的故事感动了我，”温德尔说。"从那以后，我变得更加善解人意和灵活了."

另一个年轻人也很有挑战性。

“我也有一个无家可归的学生，”她说。“我从家里收集了一些玩具，准备送给这个每天“回家”去酒店房间的学生。”

# **生死攸关的事情**

瓦莱克面对了太多的戏剧。

“除了我们的态度、对种族的看法、刻板印象、经济背景以及我们如何看待学业成功，同情心还关注这些问题的核心，”他说。

当悲剧多次发生时尤其如此。

“在我的一个七年级学生不幸去世后，我们的前校长让我们学校在体育馆聚会，一起分享和哭泣，”Varlack 说。

“伙计，我教书的头三年，每年都有学生通过，”他说非常清醒。"

这些老师努力让同情心成为学生整体教育经历的一部分。

一般来说，不要严格照本宣科。把个性和灵魂融入课程。从 A 到 z，生活并不是一条直线。深入挖掘，将生活融入课程中。

“我一直告诉我的学生，是的，我想让你学习所有这些东西，并且擅长阅读、写作和数学，”温德尔说。"最重要的是，你是一个为他人做好事的好孩子."

# **你有一个朋友**

她分享了她在学校开始的一个项目的[新闻视频](http://www.yourcentralvalley.com/news/playgrounds-pals-looking-out-for-one-another/1101475608)。

“我们希望培养三年级学生的同情心，因为他们意识到并不是每个人都有朋友(T4)，”温德尔说。

Varlack 强调为每个人提供平等待遇的项目。

“停止创造一个孤立学生的系统，因为我们认为他们应该基于种族、经济背景和学术水平而被隔离，”他说。“学生在校外可能已经很艰难了。在我们的四面墙里面应该只有同情。”

Varlack 不容忍学术人员捉襟见肘的观念。

“有了我们学校为数不多或很多的伟大教育家，我们肯定能接触到每一个学生，”他说。

# 未知需求

分享同情心可以改变学生的生活。

你永远不知道学生的心理需求。一个同情的举动可能是他们那天唯一真正的人与人之间的联系。老师能给他们希望，仅仅是因为他们在乎。

“对我来说，最令人兴奋的部分是我不知道，”温德尔说。“这就是为什么我们需要继续保持同情和爱。

“你永远不知道它对生活的影响，”她说。"这可能就是所有的差别."

在米勒的学校里，最能为学生服务的人被选拔出来。

“作为一名中学校长，我们尽了最大努力来确定大楼里每个孩子信任的成年人，”他说。“我尽一切努力与最不期待也最需要它的学生接触。”

瓦拉克知道手势会持续一生。

“同情心可以改变生命的轨迹，”他说。“一刀切的教育制度无法满足所有学生的需求。

“我们必须退后一步，重新聚焦，用同情心来沟通，”Varlack 说。

# 带来好的感觉

在教育工作者的生活中分享同情心可以磨平粗糙的边缘。与其对意想不到的变化愤愤不平，开放的心灵和思想会带来好的感受。

米勒说:“这肯定会让事情变得更难，但事情本来就应该这样。”“这是最需要我们的地方。

“我一直很感激那些信任我的老师，他们分享了他们的个人奋斗，并给我机会向他们表达同情，”他说。

瓦拉克敦促老师们随时准备满足他人的需求。

“这可以帮助他们在午餐时间不关门或独自坐在教师休息室里，”他说。
