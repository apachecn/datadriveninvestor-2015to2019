# 韩国庞氏骗局

韩国的投资者大多是老年人，他们被承诺购买的 M-token 价值将增长 600%，被骗走 1870 万美元。5.6 万人被骗。韩国当局在应用机器学习算法检测庞氏骗局实施者常用的关键词和短语后，发现了这一欺诈行为。

# 加拿大的 ICO 欺诈

凯文·霍布斯和丽莎·张在不列颠哥伦比亚省面临民事资产没收，因为他们被指控诈骗投资者 3000 万美元。在推出燃料代币后，他们告诉投资者，他们的加密货币将大幅升值。他们被指控使用投资者资金购买 SUV、一栋联排别墅和赌博。

# 台湾 5100 万美元被盗

几名台湾男子被指控诈骗 1000 名投资者 5100 万美元。投资者被承诺 335%的回报，但他们什么也没得到。责任人将面临数年监禁。为了应对这一庞氏骗局，台湾政府对《洗钱控制法》和《防止资助恐怖主义法》进行了修订。

# 阿米特·拉坎帕尔在孟买诈骗案中被捕

Amit Lakhanpal 是被指控盗窃 7160 万美元的四名男子之一，这是 ICO 骗局的一部分。为了吸引投资者，据说他和他的合作伙伴夸大了他的加密货币的价格，并声称迪拜王室支持他的项目。

# 索菲亚诈骗案中被盗 1.14 亿美元

作为大规模调查和诱捕行动的结果，欧洲警方发现了一个跨越捷克共和国和保加利亚的犯罪组织，该组织从事二元期权和加密交易平台的非法操作。这些加密平台的用户被承诺获得高额投资回报，但却被诈骗了 1.14 亿美元。

# 不要陷入加密欺诈

庞氏骗局、ICO 欺诈和类似的犯罪活动通常为投资者提供不切实际的高投资收益率，而经营者通常会要求投资者帮助他们吸引更多受害者以换取高额佣金。在寻找投资加密货币时，运用你的最佳判断力，这样你就不会被利用。
