每当学生选择学习一门新的编程语言时，他/她的脑海中就会出现许多问题。特别是对于那些遵循传统方式学习 C、C++然后试图找到下一步的学生。这可能是你可能做出的最重要的学习选择，因为它可能会成为你未来的工作领域，你会迷失在其中。在这一点上，我面临着困难，我心中有几个问题。

我将学习什么语言？

选择好学什么之后，我会怎么学，是去看书还是去网上看？

我应该参加网上课程还是从随机资源中学习？

我什么时候才能学完这个？

当然，找到答案对我来说很难，但我最终做到了。我只是分享我找到这些答案的方法，这样可以帮助你们找到答案。

# **选择你想学的，而不是别人告诉你要学的**

当你试图找到下一个大的步骤时，你会有很多选择。人们会建议一些领域，声称在那个特定的领域有很多机会。现在我不会说完全不要听他们的，听听他们说什么但是在选择你想进一步工作的领域之前要自己做研究。确保你喜欢它，并且想了解更多。只有这样，学习才会起作用。不要仅仅因为有人建议或者仅仅因为你的朋友在做而占用一些随机域。考虑所有的选择，但在选择之前要确保你真的想要它。

一旦明确了领域，你将不得不选择语言。再一次，你有互联网来做你的研究，找到每一个你应该学习一种特定语言的理由，以及为什么不学，那种语言的利与弊。大多数语言都支持多个领域。选择一个最适合你的兴趣，并且是或有可能成为一个大规模的趋势。这里人们会有不同的意见，考虑他们，但不要轻信他们。

# 书是好的，但是要经常在网上接触最新的更新。

书籍给人一种随时打开来学习的快乐。你不需要互联网连接，它们来自一个可信的、经验丰富的作者，他实际上已经使用了这些技术。你总会在官网上找到书籍的推荐。有值得信赖的作者，同时也有每天尝试新事物的技术人员和自由职业者。所以，如果你是一个爱看书的人，那么就以这本书为标准，同时一定要在网上查阅你所学的一切。你可能会发现许多不同的实现方式，更简单的解释，更实用的例子来帮助你更好地理解概念，最重要的是，一本书不能立即更新，所以继续浏览

如果你不像我一样是一个爱读书的人，你唯一的选择就是要么找一个标准的团体，要么找一个人或一个社区来发布在线内容分享他们的知识。搜索互联网上的免费资源，并在此基础上进行构建。尝试开源社区博客和其他自由职业者博客作为搜索的线索。如果什么也没找到，剩下的唯一选择就是参加付费在线课程。在选择这个选项之前，请彻底搜索。

# 文档是关键

可以从任何资源中学习，但是验证这些学习和更好地理解概念的最好方法只能是在语言的官方文档中引用相同的内容。无论是书籍、免费学习资源还是在线付费课程，一种方法、一种新的数据类型或一个库都可能是你在该课程中构建的特定内容。请务必立即、简单地查看特定新概念的一般视图，最好的方法是参考文档。

# 学习语法并不意味着学习语言

许多人，尤其是学生犯这个错误。我也做了。在文本编辑器中输入一个特定的语法将会得到这个输出。这到底是怎么不学呢。实际上任何人都能记住它。正确的方法是理解语言的基础和操作的基本单元，语言的不同方面如变量、迭代、控制流以及高级语言到机器语言的转换是如何发生的。

# 不要学着做项目，要学着用项目

总是试图通过创建简单的程序来学习，而不仅仅是通过键入和执行教程中给出的例子。试着想出一个创造性的方法来应用不同的语法，并实现它，以获得更好的理解和满意，你会很好。将这些代码保存在一个单独的地方，因为当你遇到必须回顾和重新评估所学内容的情况时，它们可能会非常有用。这很重要，因为你写的代码越多，你就越强大。

# 完成学习…

嗯，这是计算机科学，不，你不能完全学到东西，检测错误并修复它们，需要更高的性能和其他因素导致更多的更新，这一直在重复。能做的最好的事情就是在基础上变得强大，然后你可以赶上其他的。

# 结论

所有这些行动让我清楚地知道我应该学什么和如何去做。最终以这种方式进步帮助我找到了我的答案，我会推荐它，因为这些步骤对我在大学一年级的学习有很大的帮助。灵活选择您的技术组合，因为您想这样做而这样做。祝一切顺利，继续学习。
