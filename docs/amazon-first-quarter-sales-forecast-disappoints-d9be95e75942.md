# 亚马逊第一季度销售额

> 原文：<https://medium.datadriveninvestor.com/amazon-first-quarter-sales-forecast-disappoints-d9be95e75942?source=collection_archive---------29----------------------->

![](img/b5f2c7a75aa53e73aef3114165fb2a23.png)![](img/81f5479522c2d488ad67d2054741e8c3.png)

亚马逊似乎出了什么问题？预计这家科技巨头将公布今年第一季度糟糕的销售数据。亚马逊将在 Q1 实现增长，但销售数字将低于预期。因此，投资者这些天在交易 AMZN 股票时似乎相当悲观。亚马逊最近看到其股票价值下跌。

一些专家用该公司在海外面临的问题来解释其困境。例如，在印度，由于对外国公司在该市场销售商品的新限制，该公司不得不停止销售一些商品。其他专家指出，在美国国内，竞争对手在报复亚马逊，亚马逊会停止统治在线零售吗？

不过，亚马逊的粉丝不必担心。该公司面临的问题可能是暂时的。该公司的基本面依然强劲。正如我前面说过的，今年第一季度的销售数字可能会高于去年第四季度。只是在线零售可能不是公司短期内的主要增长动力，其他部门如广告可能会取而代之。
