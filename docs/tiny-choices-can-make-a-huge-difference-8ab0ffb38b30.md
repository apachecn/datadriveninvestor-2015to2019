# 微小的选择可以带来巨大的不同

> 原文：<https://medium.datadriveninvestor.com/tiny-choices-can-make-a-huge-difference-8ab0ffb38b30?source=collection_archive---------16----------------------->

在混沌理论中，著名的“[蝴蝶效应](https://en.wikipedia.org/wiki/Butterfly_effect)说的是一个系统中一个状态的微小变化会导致后面一个状态的巨大差异。

一个变量的微小变化会对整个系统的结果产生相当大的影响。

你可以在《T2:城堡里的男人》系列剧中看到这一幕。它假设纳粹德国赢得了第二次世界大战，然后征服了大半个美国和世界。虚构的纳粹控制的世界和我们现在的世界之间的差异是巨大的！唯一的假设是我们的世界历史中的一个变量发生了变化:如果纳粹赢得了第二次世界大战呢？

我也在读关于苏巴斯·钱德拉·鲍斯的书。在维基百科的一系列文章把我带到他的页面后，我上了他的页面。我们知道一些他在学校的历史。但是我不知道*为什么在他生命的最后几年和日本人在一起。证据显示，他与轴心国结盟，并试图获得日本帝国的帮助，以获得独立的印度。但是看起来他对他们的独裁政府很满意。*

因此，这意味着，如果他设法取得了上风和权力，他将成为一个独立的印度的最高领导人。尽管它可能会在他的独裁统治下衰落。也许，有一天，在这个交替的宇宙上，会出现一个类似《高堡奇人》的系列。可能已经有一个系列了，我也不知道。

我要带你去哪里？

我想说服你，你每天做出的选择，即使是无关紧要的，也能给你的生活带来巨大的变化。你可能在等待一个改变生活的决定，这个决定会让你在生活中得到任何东西。但最有可能的是一系列微小的选择，这些选择从表面上看似乎无关紧要。

那么，你在日常生活中没有做出哪些你认为会产生巨大影响的小选择呢？
