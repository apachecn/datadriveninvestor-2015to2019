# **隐私币的注意事项**

Monero 和 Dash 正在争夺市值最高的隐私币的桂冠。这些项目的成功在一定程度上取决于它们所吸引的巨大社区，但不用说，这两个协议都提供了有效的隐私解决方案。然而，没有完美的隐私硬币。

**Dash:** Dash 的 masternodes 负责各种角色，包括备受期待的 PrivateSend。主节点本质上代表用户充当混合器；因此，一个关键的警告是，负责混合的人可能会滥用权力获取交易各方的信息。此外，Dash 不使用零知识证明；因此，交易链接不会中断。

**Monero:** Monero 利用一种独特的基础设施，集成了密码笔记和环签名。该网络能够在不需要混合器的情况下为交易提供默认隐私，Monero 还确保交易金额保持隐藏。然而，Monero 也没有打破交易联系，只是模糊了它们。Monero 面临的另一个主要挑战是交易量非常大，因此需要很长的等待时间来处理。

可以说，针对 Monero 的最有力的断言是，网络用来隐藏交易价值的手段也阻碍了供应审计。因此，导致过量硬币铸造的网络利用可以不被注意。

**Zcash:** Zcash 利用了 zkSNARKS，这是一种加密机制，由于其新颖性，目前仍处于试验阶段。然而，底层技术允许 Zcash 在交易中提供极端的匿名性，包括完全打破交易链接的能力。此外，网络吞吐量比竞争对手更具可扩展性。

Zcash 的缺点是，zkSNARKS 的先进性意味着只有少数人能够解决系统的先进性问题。事实上，交易值的匿名性可能会导致潜在的漏洞，从而导致未被注意到的过量硬币发行。事实上，[这样的场景](http://fortune.com/2019/02/05/zcash-vulnerability-cryptocurrency/)已经发生了。

**Zcoin:** Zcoin 采用 Zerocoin 协议，该协议采用久经考验的 RSA 因子分解。这样做的好处是，这是研究最充分的技术之一。尽管如此，Zcoin 交易的绝大部分并不是匿名的。这是因为 Zcoin 交易可以通过 Zcoin 或协议的基础硬币来完成。Zcoins 需要从 Basecoin 铸造到 Zerocoin，只能以固定面额铸造，并且对铸造的财政激励有限。因此，很大一部分网络用户只是选择交易不匿名的基础货币。

虽然 Zcoin 可以提供匿名加密货币，但网络的交易量主要是可见的基础货币。

# **没有完美的硬币**

每个隐私硬币都有一定的利弊。虽然 Dash 是最具可扩展性的网络之一，但它集成了 masternodes 进行混合。Monero 试图以极其缓慢的交易为代价来规避 Dash 的警告；它也没有提供审计供应的手段。

Zcash 是一个先进的协议，它面临着 Monero 的关键困境:没有办法审计供应。因此，存在时间最长的 Zerocoin 协议通过其可审计的供应在网络中提供了更多的可靠性。然而，Zcoin 网络交易证明，Zerocoin 协议的当前应用未能用于匿名。

因此，PIVX 的创始人，另一个领先的隐私硬币，旨在建立在现有的隐私硬币上。Veil 项目整合了 Zerocoin 协议和 Monero 的基础硬币 RingCT 机制。

# **追求最佳隐私币**

由资深开发者和区块链建筑师组成的团队“面纱实验室”旨在推出最实用的隐私硬币“面纱”。Veil 将利用 Zerocoin 协议，而不是涉足可能将网络置于风险之中的实验技术；然而，Veil Labs 的存在将确保 Zerocoin 协议现有应用的所有警告都得到解决。

首先，通过将 RingCT 应用于基础硬币，面纱网络内的所有硬币将具有隐私特征。因此，即使用户避免用基础硬币铸造面纱，他们也可以用基础硬币匿名交易。然而，目标是确保铸造不被视为一种负担。为了实现这一目标，Veil Labs 有一个雄心勃勃的目标，即允许用户进行多块交易，以解决必须铸造零硬币并以固定面值进行交易的货币问题。

虽然协议改进是 Veil 存在的一个关键支柱，但该项目的最后一个里程碑是提供一种隐私硬币，具有大众愿意接受的用户体验。如果网络是可扩展的，并且容易被大众使用，那么不管加密用户是否关心隐私，协议都是一种方便的交易方式。在大多数区块链项目中，解决用户体验是一个被忽视的关键因素；因此，维尔考虑向大众推出一款区块链友好型产品，这是区块链发展迫切需要的一个关键转变。
