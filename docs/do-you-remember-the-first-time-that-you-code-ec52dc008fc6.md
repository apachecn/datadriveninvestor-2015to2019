我自己也在思考这个问题，我记得我第一次编码的时候，我遇到了一个很大的问题。我在中学，我喜欢玩胫骨(一个在线游戏)，所以我搜索并试图建立一个这样的游戏。很长一段时间后，我们终于创造了一个游戏，我们玩了很多，我们改变了数学，葡萄牙语来玩游戏。无红利

我又在思考了…我听到了很多我的朋友告诉我他们为什么成为软件工程师的不可思议的故事。他们中的大多数人真的热爱他们的工作，我也是！但是有时候我们会忘记加入这个领域的动机。

当我在做我喜欢的事情时，我必须提醒自己停止工作，因为一旦我开始工作，我就不想停下来。但是当你不相信或不喜欢你想要的，而你只是继续做，你就不会有同样的精力…你明白我的意思吗？现在我感觉很幸运，因为在我的工作中，我们可以选择我们想做的项目，我最近开始做一些开源项目和私人项目，对此我非常兴奋。
