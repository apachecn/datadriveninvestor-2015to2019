# 不，这不是钱的问题。

如果你是一个成功的投资者，很有可能你的大部分回报来自于一手的投资。有些人认为这个想法是帕累托原则——你 80%的结果来自你 20%的活动，反之亦然。在我看来，我喜欢考虑投资一个有头有尾的游戏，也就是长尾巴。少数投资者从满手的投资中获得巨额回报，这些投资来自于在正确的时间、正确的地点和正确的思维模式(或不正确)，很少全是智力。

> 这里有一个问题:与投资不同，幸福不应该被追求。它自发地出现，在随机的时刻——在工作或休闲的时候。

所以，我为什么首先提出帕累托原则——你应该满足于投资和幸福，尽管很多事情都没有成功。学会体验人类情感的全波。

生命中只有满满一手的东西才值得。剩下的就稍纵即逝了。如今，我日复一日地冒险、学习和工作，甚至在旅行中——这给了我最深刻的活着的感觉，对 26 岁的我来说，我对此很满意。
