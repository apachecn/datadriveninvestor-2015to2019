# 拥有数据—让您的用户能够轻松使用数据

“给一个人一条鱼，你可以喂他一天。教他如何钓鱼，你就养活了他一辈子”这句话在我写这篇文章的时候一直在我的脑海里萦绕。

几个月前，Zalando 的组织发生了巨大的变化。我们的营销情报团队蒸发了，我成了个性化营销团队的一员。我的职位从数据员变成了营销产品经理。六个月的学习和领悟，我有话要说:玛丽亚·布里吉达·德勒纳尔迪斯，对不起！

在成为营销团队的产品经理的过程中，我认识到自己失败了。作为一个有强烈意见的数据人，我没有以应有的方式看待我的利益相关者。我徒劳地认为我知道我的利益相关者应该如何最好地消费他们的数据。虽然我没有完全错，但我应该用不同的方式去做。

布里吉达是我最大的数据消费者。她每天都依靠我的报告来了解她的频道进展如何，而我让她失望了。布里吉达不断打电话给我，抱怨数据是错误的，但作为一个“好”的 IT 人员，我不同意，并向她展示了她的错误。问题是，她无法使用这些数据，因为我没有向她解释如何使用这些数据。最重要的是，我建立了一个固定的报告，并强迫她使用它，只是因为管理层也在使用它。

因此，在做了六个月的营销经理后，我仍然可以说我的固定报告非常好看，它从 C 级到在职学生都是如此。然而，在连接到不同数据源的同时构建自己的报告和可视化的能力帮助我发现隐藏的问题并与我的团队分享。

想想一个坏的营销工具，它不断增加合作伙伴的出价，尽管在 800 欧元的预算下出价 80 欧元是没有意义的。发现这种错误的最快方法是通过内部仪表板来监控活动。

我学到的最重要的事情是，数据提供商不应该只关注数据质量和访问，他们还应该与他们的利益相关者合作，建立信任，并确保他们也理解数据。他们应该站在客户的角度去理解他们有什么问题。我了解到这个大报告只应用于调整每个人。利益相关者不应该被迫使用报告。相反，他们应该构建自己的仪表板和可视化工具来监视和控制他们的活动。

作为数据使能者和数据控制者，这是一个需要改变思维的领域。我们应该从认为人们太愚蠢而不能处理数据，到让人们更多地处理数据。

我们应该给他们合适的工具，教他们我们做什么，这样他们也可以根据自己的需要使用它。

这就是我向利益相关者推荐 Power BI 或 Tableau 的原因。任何知道 Vlookup 和一些旋转的人都可以开始使用它来轻松地探索大量的数据。

现在，去找你的布里吉达，请求她的原谅，因为你一直都是错的。你没有用数据来增加她的安全性。你没有帮助她发展分析思维。你把她关在花园里，强迫她用你认为正确的方式来展示数据。
