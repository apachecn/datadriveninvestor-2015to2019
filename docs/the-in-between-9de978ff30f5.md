# 大多数人认为他们乐于接受改变。

我们走来走去，说着诸如“改变是不可避免的”或“改变或死亡”之类的话，但请某人换办公室或使用新的停车位，所有这些头脑层面的知识都飞出了门，根深蒂固。顺便说一句，我对此感到内疚，在抵制改变或避免所有抵制之后，需要这么多的日常反思才能自我纠正(这还没有发生)。

人们倾向于停留在两者之间。我们的价值很大程度上来自于我们做过的事情，我们拥有的东西，我们在生活中所处的位置——很难把这些都抛到九霄云外，去思考接下来会发生什么。我认为关键是要认识到，无论你选择做什么或在生活中做什么，你都在付出代价。

> 明确我们要付出的代价，决定这是否是我们可以接受的，然后选择继续前进或改变策略，这是创造我们想要的生活的非常有力的第一步。

如果你追求的是安全感，或者至少是一份可预测的工作时间表，那么你很可能会找到一份很棒的工作，并一直干下去。你已经决定了你愿意付出的代价是自主权。同样，如果你是一个热衷于创造遗产的企业家，那么你的代价就是安全和稳定。

知道不管我们在生活中处于什么位置，我们都在付出代价——让我们能够做出更好的生活决定。假装挣工资或开发票是没有成本的，这意味着你没有清楚地看到所有的选择，故意选择你想要的。清楚我们愿意付出的机会成本意味着我们可以在生活中以权威和清晰的方式谈判。

# 对我来说，介于两者之间的是对选择和故意牺牲的不自觉。认为自己一直过着人生中最好的生活是天真的。现实是，我们在做出最好的选择和目前最好的选择之间摇摆，而这些选择是我们经常被迫做出的。

了解这两者之间的区别以及我们在任何给定时刻所处的位置，意味着我们可以重新与我们的目标保持一致，并使我们愿意为成功付出的代价更符合我们可以接受的程度。

生活就是改变，就是选择。真实地面对这两者是创造非凡生活的起点——不管我们每个人看起来是什么样的。
