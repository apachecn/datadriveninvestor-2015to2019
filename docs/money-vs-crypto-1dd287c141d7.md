# 钱或触摸我，你会感受到真正的力量

纸币和硬币发出一种力量的声音来建立任何可能的想法，任何可能的愿望。但是它的效果和几年前一样吗？现金的时代不是已经过去了吗？

只有 8%的货币是实物货币。以后当你说钱的时候，你指的不是现金，而是你银行账户里的钱。如果你确实对通过银行和网上银行系统管理的资金感到安全可靠，那你为什么对信任数字钱包犹豫不决呢？

如果你看不到真正的钞票，为什么不通过区块链系统进行交易。

如果你不碰美元、欧元、卢布、日元，你为什么犹豫不碰比特币。

这难道不意味着“钱”这个词已经过时了，现在是数字货币的时代了吗？

几年前，数字这个词会带来很多恐慌和怀疑，但现在你们都在使用它，无论如何，它不是你生活的一部分。**数字是你的生活方式。**
