我们的目标真的会阻碍我们的生活吗？

这是一个真实的人问的真实的问题。

令人震惊。

如果你想继续设定“安全”的目标，今天的文章不适合你。

它不适合你，如果你想为那些让你问像上面这样的狗屁问题的选择辩护的话。

如果你只想有足够的钱，做足够的事，去付账单，去度年底的假期，然后明年再做一遍，那肯定不适合你。

我没和你说话。

相反，

我只和饥饿的人说话。

那些一直坐在生活的教室里，学习艰难困苦，忍受他们真正渴望的东西和他们认为他们可以拥有的东西之间的内心斗争的人。

我相信目标！

他们继续驱使我。

他们继续改变着我，我的生活，我的目标。

我的朋友，这不是因为你设定了平庸的目标。

事实上，

我因平庸而窒息。

每当我陷入别人荒谬的“真理”中，即什么对我是可能的和不可能的，什么对我是负责的和不负责的，我就感觉我的灵魂在收缩和死亡！

去他妈的！

那些让我冒冷汗的目标，

那些让我浑身颤抖，肾上腺素在我的血管里涌动，

泪水在内心悄悄形成，这是我真正恐惧的唯一可见迹象，

这些是我唯一承诺的目标。

他们是我在想清楚之前就答应的人。

因为它们直接来自我的灵魂。

她知道…

她知道我需要在哪里成长。

她知道我在哪里没有充分展示我的能力。

她知道我正蹲在我头脑制造的玻璃天花板下，是时候把它粉碎了。

她知道什么时候是时候。

当我放慢脚步的时候。

敷衍了事，但缺乏行动背后的情感动力。

让我们明确一点，目标与打勾没有关系。

不是越过终点线让别人为你鼓掌。

这不是小孩子的问题。

我提到的目标，我设定的目标，都是个人的。

它们只关乎成长。

其他的一切都只是装饰品。

这些目标怎么会阻碍你的生活呢？

所以这里是我个人的一些目标标准。

有些会产生共鸣。

有些会触发。

你要怎么做取决于你，我的朋友。

我想说的是我们的时间不多了。

我们不是生来就在脚底纹上了有效期。

那我们为什么要退缩呢？

# 1.我的目标不期而至。

我从不考虑我的目标。我不会去找他们。

它们是原始的，它们是被引导的，当我的灵魂准备好让它们出现的时候，它们就出现了，就像被施了魔法一样。

有人会说一句话，我的心脏就停止跳动。我会看到一个视频，我的嘴开始流口水。我的笔会在页面上流动，我会站到一边，睁大眼睛看看会出现什么。

# 2.我从来没有为我的目标做好准备。

我他妈的不知道该怎么实现它们。我甚至不再怀疑了。我没有看到一个梦，并按下暂停键，这样我就可以弄清楚是怎么回事了。

他们总是远远超过我目前的能力。

它们总是比我想象的要大得多。

我已经不在乎了。

我只是承诺。

就是这样。

我知道，因为我是一个说话算数的女人，一旦我做出承诺，一旦我对我的灵魂说是，这就是他妈的成交。

在 DNF 之前死亡。

# 3.如果我不需要引进大人物，那就不是我要设定的目标。

我的目标需要支持。他们需要指导。他们需要专业知识，而我没有。

我所有的目标都需要导师和教练。

如果我自己能想出来，如果我有时间自己想出来，那就是一个低于标准的目标。

走开。

我的目标需要我全身心的投入。就像一个嫉妒的情人，他们占用了我的资源、时间、注意力和精力。

他们要求我对他们进行大量投资，从而对自己进行大量投资。

当我听到人们设定目标，然后说他们会自己去实现，我知道他们仍然在玩小游戏。

# 4.我的目标需要牺牲。

一切都是能量。

一切都在一定程度上振动。

包括我。

无论当前在我周围的是什么，对我来说都是一种互补的振动。

我的目标不是。

他们在更高的水平上振动，为了让我上升到他们的频率，我必须愿意牺牲我当前振动的一切。

更低振动的想法。

更低的振动信念。

更低的振动想法。

低振动食物。

较低的振动关系。

下部振动衣服。

你觉得无所谓。你以为抓住几样东西就能逃脱。

你不能。

这就是为什么我的目标需要牺牲。

放手几乎总是充满了某种程度的悲伤。

它总是充满了恐惧。

我发现我喜欢生活中的确定性。这就是为什么我有如此严格的惯例。这就是为什么我对一些事情坚持太久。

我花了很多年才认识到，我觉得变化和不可预测性很可怕。

我花了一天的时间来为我自己重新定义那狗屎到 ***改变等于冒险*** 。

我们都知道妈妈喜欢冒险！

这无疑改变了我释放不再为我服务的事物的方法。但是永远不要认为我是他们说的那种冷酷无情，不顾别人的婊子。它仍然像狗娘养的一样疼。

人是最难丢下的。

然而，我选择相信那些真正渴望在我的空间里玩耍的人，会去做他们自己的工作来和我一起上升。

至于其他的，我尊重他们的旅程。

选择的自由亲爱的。

# 5.我的目标不允许休假。

我的目标既大又复杂，它们要求我把它们分成更小的部分，设定子目标，然后每天都做这件事。

即使是今天。

当天气变成奶油冻的时候。

倾盆大雨。

呼啸的风。

我得去骑马了。

因为我周日要比赛。

尽管这是一场米老鼠比赛，一场 C 级比赛，但这是我 2020 年 A 级比赛旅程的第一步。

一切都很重要。

这不是错过一节课的问题，而是错过一节课对我的影响。

它说我是个娘娘腔。

它说我不可靠。

它说我不是那个需要跨越最终终点线的人。

它说我对自己不守信用，对我来说，这根本不是一个选项。

很久以前我就决定说我想说的话，并且说到做到。

即使人们不相信我，因为大多数时候，他们很久以前就不再相信自己了。

我的生命，我的责任，我的承诺，我的荣誉。

我的观点是，只要你设定了你的灵魂目标，它们就永远不会阻碍你的生活。他们将永远推动你前进，更高，更大，更好。

如果它们不是，如果它们不需要严肃的承诺和牺牲，不需要克服障碍，不需要面对你最大的恐惧，它们就不是你应该设定的正确目标，亲爱的。

不要一次设定一百个目标。

停止设定你可以自己实现的目标。

停止设定让你今天和昨天一样的目标。

停止设定美好的目标。

停止设定别人认可的目标。

停止设定平均目标！

否则，就满足于过普通人的生活。

就这么简单。

只有死亡不可避免。

繁荣永远是勇敢者的选择。
