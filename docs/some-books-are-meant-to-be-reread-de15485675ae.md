## 因为我们忘记了

在过去的几年里，我读了大约一百本书，但我只记得 1%的内容。这也太夸张了。我从未重读一本书，但我知道我必须重读。

这是因为无论作者一开始想告诉我们什么，我们都没有领会。许多读者坚信，每次重读一本书，甚至是一篇短文，都会有新的收获。

重读加强了我们对记忆内容的把握，但它也给了我们机会去理解第一次阅读时没有理解的东西。 [**黑天鹅**](https://www.amazon.in/Black-Swan-Impact-Highly-Improbable/dp/0141034599) 对我来说就是一个例子。我从那本书中学到了很多有趣的想法，但是我知道我并没有理解更多的东西。当我读它的时候，我没有必要的背景和知识去理解它。总有一天我会重新读一遍，让我大吃一惊。

当我们阅读我们知识范围之外的非小说类书籍时，很有可能我们没有理解所需的所有背景资料。但是你不能在看书之前知道。一旦你知道你可以从别处获得更多的知识和智慧，然后用新的眼光重新阅读这本复杂的书，你会发现你会更加欣赏它。

这看起来像是浪费时间，对某些书来说，可能是这样。但有些宝石值得反复阅读。
