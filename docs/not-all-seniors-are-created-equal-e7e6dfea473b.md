# 无效资历的原因

根据我和客户的经验，这些都可以归结为一个自主性问题或者是你职业生涯中的一个问题。

适当的自主性是至关重要的，我经常把它作为实现团队交付的方法的三个部分之一。你的员工，尤其是年长者，应该拥有完全的所有权，并接受他们对事情如何运作有发言权的观念。当他们不同意时，也不应该害怕直言不讳。这些都是文化问题，通过适当的引导，领导层可以很快解决这些问题。

不幸的是，另一个原因更难解决。如果你的招聘流程或职业阶梯的提升有所欠缺，导致人们被放到错误的“盒子”里——你的日子会不好过。聘用某人担任职员/校长或其他高级职位，却发现他们并不积极参与团队建设，这是一个严重的错误。

你仍然可以纠正错误，但这意味着重新审视你的招聘流程、期望值设定和定期指导流程。

回到我们之前的练习:你认为你的进展如何？如果你对你脑海中的画面不满意，记住这一切都从你开始，你有能力改变它。
