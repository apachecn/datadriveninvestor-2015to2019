## KYC/反洗钱

要提供大多数金融产品，您需要遵守了解您的客户(KYC)和反洗钱(AML)规则。这将涉及要求用户提供个人和财务信息，并验证其真实性。这些提供者将验证收集的信息所需的大部分工作自动化。

*   身份验证——他们上传的护照或驾照照片是真实的吗？ [Trulioo](https://www.trulioo.com/) 和 [Jumio](https://www.jumio.com/) 将为美国和国际申请人处理检查。
*   准确性和监视列表扫描—他们提供的地址真的是他们的吗？他们的法律地位是否与洗钱或国际观察名单无关？Blockscore 通过它的 API 处理这些检查。

## 账户检查

在接触客户时，您通常希望确认并联系他们拥有的金融账户。也许你想确认余额或主动将资金转入他们的新账户。

*   [格子](https://plaid.com/)
*   Yodlee

## 检查

想开一个支票账户吗？跳过寻找持有资金的银行合作伙伴，让你拥有客户。这些 API 使您能够跳过合作和管道跑腿工作。

*   [SynapseFi](https://synapsefi.com/)
*   [拉皮德](https://www.rapyd.net/)
*   [债券](https://www.bond.tech/)(尚未完全上市)

## 信用卡

也许你已经掌握了传统信用卡，并认为你可以进入交换费的流动。也许是给青少年的卡或者给 T2 的卡来控制你的开销？避免前期费用和说服发卡合作伙伴，而是使用这些 API 之一。

*   [条纹](https://stripe.com/issuing)
*   [SynapseFi](https://synapsefi.com/issue-cards)

## 股票交易

现在，你已经让数百万人直接存入他们的工资支票，并用你的信用卡消费，让我们为他们提供一些投资工具，让他们的钱发挥作用。当然，他们可以开一个电子交易账户，但是在他们已经用于个人财务的账户中这样做不是更容易吗？你可以花几个月、几十万美元来启动一个经纪自营商，或者利用这些 API 立即开始。

*   [羊驼](https://alpaca.markets/)
