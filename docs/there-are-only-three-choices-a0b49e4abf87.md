# 只有三个选择

> 原文：<https://medium.datadriveninvestor.com/there-are-only-three-choices-a0b49e4abf87?source=collection_archive---------32----------------------->

[![](img/ae2aace727e7d3a88474369e0a8d4301.png)](http://www.track.datadriveninvestor.com/1B9E)![](img/e1327dec57a018a9b78adee958c2e35f.png)

> 如果你试图在一份工作、一段感情或一项任务中取得成功，你要么前进，要么落后，要么停滞不前。只有三个选择。”

这是赛斯·戈丁的至理名言。

虽然很明显这是仅有的三种选择，但当我们实际评估我们目前正在做的事情处于这三种状态中的哪一种时，这一点就一点也不明显了。

忙于工作——检查邮件、回复空闲时间的通知等——很容易让人产生前进的错觉。仅仅因为我们在做占用我们时间的事情，并不意味着我们在前进。

这就是我如此热衷于跟踪指标并以 OKRs 为生活准则的原因(约翰·杜尔登的《衡量重要因素》是一本关于这个主题的迷人书籍)。当我们清楚地认识到我们将如何衡量某件事情的进展，并开始定期衡量它时，我们就可以明确地回答我们是在前进还是停滞不前还是落后了。

正确识别我们处于三种状态中的哪一种状态的直接后果是，我们将被提示采取行动。如果我们正在前进，我们可能希望加快速度。如果我们停滞不前，我们可能会尝试不同的方法，如果我们落后了，我们可能会停下来，重新思考我们正在做什么，以及我们如何实现它。

只有三个选择。只有三个州。我们有责任确定我们所处的位置，并采取相应的行动。
