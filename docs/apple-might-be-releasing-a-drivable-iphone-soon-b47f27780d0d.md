# 不过，我们知道你在想什么:无人驾驶汽车？科幻小说。

与此相反的是。科学*事实*。不过，说真的，这看起来一点也不像《追忆似水年华》中的经典片段，影片中你渴望去火星旅行，只是为了满足你奇怪的好奇心:

当然，别再吃零食了。让我们严肃点…自动驾驶汽车是我们很快就会看到的真正发展。事实上，我们现在已经看到谷歌的 **Waymo** 成为头条新闻，已经在亚利桑那州凤凰城测试自动化。可以肯定的是，随着新创新的不断发展，其他某些汽车公司，比如*特斯拉汽车*，可能会跳上这辆约翰尼驾驶室，自主驾驶，但事实是:

![](img/903c41b9adfa33ab62585826917b41f8.png)

苹果希望进入这个尚未开发的收入渠道，这个渠道始于电动汽车和混合动力技术。他们的自动驾驶汽车项目 **Project Titan** 可能会推动该公司超越 iPhone 进入下一个阶段。它可能是所有人工智能项目的*之母*——正如你所知，苹果的*此刻对 AI 非常感兴趣。*

然而，如你所见，该公司目前完全落后于谷歌。不太好。但是，我们在这里谈论的是苹果。尽管苹果确实从其自动驾驶汽车部门裁员 200 名员工，但事实是这次裁员只是苹果人工智能部门的重新分配。

关键词:都是关于 [**人工智能**](https://vigyaa.com/@pierre/not-savvy-with-artificial-intelligence-ai-you-might-as-well-be-00331c57/) 。如果他们做对了，我们可能很快就会看到“轮子上的 iPhone”。只是不要在两周内。
