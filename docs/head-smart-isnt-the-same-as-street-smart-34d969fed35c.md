# 让我承认:我经常相信自己比周围的人聪明得多

我也经常被这种自我谴责打耳光:“是啊，那又怎样？你用你那聪明的脑袋完成了什么？”哈，目前为止没什么重要的。

我喜欢阅读复杂的话题，如心理学、现代哲学、医学、技术发展。但我注意到，我经常向其他人吹嘘我的阅读材料有多“高级”,而不是他们选择的世俗话题。我不只是集中注意力去理解更多的主题，而是经常浏览它们去理解表面层次。为什么当然，给他们讲一些琐事也不会有什么麻烦。为了满足我的自我，我花了这么多精力却没有得到任何有意义的回报。

现在我已经厌倦了做这样的事情。通过一些深刻的回顾，我开始放下对确认和证实的需要。与其虚张声势，为什么不慢慢地、稳扎稳打地做点实事呢？期待每个人的认可是一个危险的游戏——如果我没有得到我想要的，我会非常失望。

所以，现在我的目标不仅是头脑聪明，也是街头聪明。做更多的实际行动会产生更多的经验，从而比以往任何时候都有更多的成长。尝试扩展我的视野，我的世界观，以及一个新的兴趣领域，可能会帮助我从过去的自己成长为我渴望成为的人。

我从任何领域的专家那里注意到的一点是，他们不是炫耀他们所知道的，而是越来越专注于学习——当他们不知道一些事情时，也有口才承认。

也许如果读者能从这篇看似自我专注的文章中得到些什么的话，学会放下对每个人认可的需要，将有助于你过上自己真正渴望的生活。我们，人类，对自己的生活几乎没有控制力，然而即使我们拥有如此之少，我们也能解决阻碍我们前进的问题。
