# 更快、更高效的招聘

在 [Sprint Recruiting](https://nakedhr.me/sprint-recruiting/#.XNdjzDBKiHs) 中，我提到的一个主题是更快、更高效地招聘。sprint 招聘方法的一部分是回顾，它允许我们的团队检查在 Sprint 期间什么有效，什么无效。我们将经常使用数据分析来分析候选人流程、流程中的瓶颈和其他重要数据点，以帮助即将到来的冲刺。

为了增加雇员，分析你的雇员来自哪里是非常重要的。你可能会发现，你的许多雇员来自内部推荐或其他来源，而不是在招聘栏上花费数千美元。这种分析不仅可以帮助你减少一个职位空缺的时间，还可以帮助你将花费在无用的候选人资源上的钱重新分配到更有效的资源策略上。

# 更好的雇佣

在招聘中利用数据分析将有助于招聘人员专注于为正确的角色找到正确的候选人。分析目前在招聘职位上的成功员工可以帮助招聘人员锁定摇滚明星候选人，而不仅仅是愿意做这份工作的人。

使用像 Visier 这样的工具或其他 HCM 分析工具可以帮助你收集所需的数据，以加强你的采购策略。它还可以帮助你调整面试过程，确保你和你的招聘经理问了正确的问题，以获得对候选人的正确认识。

# 提高参与度

我非常喜欢 MailChimp 和其他帮助管理内容营销过程的服务。像这样的服务的真正价值是它提供的数据分析，帮助你更好地了解你的受众。

如果你的目标是在你的招聘策略中利用内容营销，那么了解你的读者想要什么来增加参与度是至关重要的。MailChimp 和 Hubspot 提供了令人难以置信的分析，让你可以跟踪你的帖子从你的邮件列表中收到了多少点击和分享。

随着内容营销在当今的招聘战略中变得越来越普遍，你将需要确保自己是如何在招聘中使用数据分析的高手。

你不必是一个统计呆子，也能体会到数据分析给你的招聘策略带来的巨大改进。我的建议是从你想提高的领域开始，通过试错来学习。

对于我们这些热衷于这类东西的人来说，数据分析在招聘中日益增长的影响就像是对最新的网飞系列的狂热。我们就是吃不够！
