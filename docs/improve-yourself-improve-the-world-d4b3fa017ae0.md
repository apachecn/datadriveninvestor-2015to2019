你会转向另一个方向，你会寻找其他人。你完全可以找到其他人，遇见愿意见你的人。谁会听你的。谁会包括你。

你应该仔细选择那些你想与之分享你的时间和使用你的创造力的人。你不会在冷漠的人身上浪费时间。如果他们现在看不到你，如果他们此时不集中注意力——很可能他们永远也不会。

# 从自己身上付出

你会付出而不期待回报。

一个好的回报会在适当的时候到来，来自那些看见你的人，那些当你说话时倾听你的人。

当你付出时，最好的感觉就来了，你可以看到你的付出得到了很好的回报，很有用，空气中弥漫着感激之情。

也许你会体验到给予比接受更好。

# 你是个有创造力的人

作为一个作家，一个诗人，一个艺术家，你站在别人面前，日复一日地奉献自己。

作为一个艺术家，一个有创造力的人，一个作家或者一个表演者，你把自己给了别人，你让他们发现并听到你独特的声音。他们记得你独特的声音，他们期待更多。

作为一个艺术家和有创造力的人，你总是在移动。你将目光和注意力转移到新的机会和机遇上。

# 建立你的部落

去认识值得关注的人，去寻找和倾听的人，去和你同向拉动共同负荷的人。

或者去认识那些想要不同的东西的人，那些想要你认为值得支持的东西的人。向别人学习，让别人向你学习你能教的最好的东西。

在你遇到的陌生人中，在你的朋友圈中，在你认识的人中，在你的家人中——要有选择性，选择并联系那些希望你好的人。

想想那些你选择作为你的大家庭，你的部落的一部分的人。培养和你所在部落的人的关系，培养共同的兴趣，和他们见面，谈论重要的事情。

走出孤独和瘫痪的被动。和处于类似情况的人在一起。听和说，找到共同的兴趣点和生产差异。

和对你好的人在一起。一起谈论、计划、发明项目，一起帮助世界变得更好。

为你的部落朋友服务，也让他们知道你自己需要什么，他们能帮你什么。
