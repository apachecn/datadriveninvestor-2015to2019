最近，我在 Coursera 完成了 [*吴恩达深度学习专业化课程*](https://www.coursera.org/specializations/deep-learning) 系列。我希望这篇评论对那些想进入这个领域或者只是巩固你的深度学习知识的人有所启发。

![](img/f14c5fcf79e43b796af5332f1f9c9c96.png)

## 课程概述

1.  ***深度学习与神经网络:***
    在课程 1 中，它讲授了什么是神经网络，正向&反向传播并指导你构建一个浅层网络然后将其堆叠为深度网络。此外，你将学习数学知识(逻辑回归，梯度下降等。)一步步与之相关。安德鲁用一种非常简单的方式解释了数学，即使你没有线性代数或微积分的先验知识也能理解。最后但同样重要的是，它分别讨论了激活函数及其优缺点。— *最喜欢的话题:如何获得正确的矩阵维数*
2.  ***改进 DNN:超参数调整、正则化和优化:***
    在课程 2 中，我们将继续学习不同的正则化技术，以及为什么它们可以减少过拟合。除此之外，Andrew 会教你如何评估你的深度学习模型，并记下他在不同情况下的超参数调整技术(这是 DL 爱好者的黄金)——*最喜欢的主题:Minibatch 应用及其如何影响模型。*
3.  ***结构化机器学习项目:***
    在课程 3 中，Andrew 会谈到如何建立评估指标，因为最高精度的模型在日常实践中可能不是最好的。他还讨论了如何选择和分割训练/测试/验证集，以及在缺乏数据时我们可以使用的方法— *最喜欢的主题:误差分析表，该表深入分析预测结果以寻找模型调整见解。*
4.  ***卷积神经网络:***
    在课程 4 中，Andrew 将通过解释卷积滤波器、Maxpooling 滤波器等的理论和数学知识，教授什么是计算机视觉以及卷积神经网络是如何工作的。最重要的是，他会浏览著名的预训练网络的结构:LeNet5、AlexNet 的 VGG-16 和 ResNet，然后指导你利用迁移学习。 *—最喜欢的话题:图像定位和检测。*
5.  ***序列模型:***
    在最后一个课程中，首先讲 RNNs 和 LSTM，然后深入到 NLP，讲授主要概念，如*单词嵌入、Word2Vec、Bleu Score、Beam 搜索、Skip Gram 等*。最后，你可以练习如何建立一个机器翻译模型。 *—最喜欢的话题:用自制的音频数据触发检测。*

## 为什么你应该接受它

***课程设计:***
该课程旨在以简单的方式教育深度学习，以降低该行业的准入门槛，助推人工智能的发展。这五门课程是一个循序渐进的系列，涵盖了深度学习的所有基本方面，尽管你只能选择你感兴趣的课程。它绝对适合具有基本 Python 编程技能的深度学习初学者。

***编程作业:***每周讲座后的程序作业都很实用，尤其是它要求你把课上学过的每一个函数都编码出来，而不是简单的调用框架。例如，它需要你编码向前&向后传播，梯度下降，分裂小批等。在 Python 中。虽然这并不是一个困难的任务，可以让你成为一个编码老手，但是这个过程可以真正帮助初学者理解它背后的机制。而对于 DL 划分器来说，这是巩固你的神经网络知识的好机会。最重要的是，所有的作业都是自动评分的，所以你可以马上得到结果并尽快进行。最后，课堂论坛会提供你完成作业所需的一切。代码的荣誉阻止学生在论坛上发布实际的代码。换句话说，你必须自己完成所有的作业。这对初学者来说可能很有挑战性，但你会在课程结束时得到回报。

***深度学习大侠访谈:***
尽管课程内容很棒，使我们能够构建和训练深度学习模型。我们都知道 DL 应用程序的广泛变化需要我们重复无数次实验，才能将模型从实验室带到日常实践中。安德鲁和嘉宾包括杰弗里·辛顿、彼得·阿比尔、伊恩·古德费勒等。分享了他们从工作中学到的很多经验，以及他们在行业中的职业道路，这对我们来说是宝贵的。

***课程认证:***
在你完成课程后，它会颁发 5 个课程认证和一个深度学习专业化认证，这些认证可以直接附加到你的 Linkedin 个人资料中。对于那些想改变职业道路的人来说，我可以说这门课程真的可以让你获得你所期望的知识和权威的认可。

***课程工作量及价格:***
每门课程的教学大纲平均为期四周，要求你每周投入 2 到 4 个小时。对于有全职工作的人来说，工作量一点都不大。课程费用仅为每月 49 美元，7 天免费试用，可以说是我上过的最便宜的 MOOC 课程之一。

## 结论

学完这门课程后，我可以预见越来越多的 DL 人才会涌现出来，因为 DL 知识使我们能够深入到我们感兴趣的主题，并将我们与这个行业的入口联系起来。我会向所有对深度学习感兴趣的人推荐这门课程，不仅是对初学者，也是对在这个领域有知识的人。知识巩固总是好的，可以教会你新的东西。如果你不想错过这十年的人工智能趋势，现在就开始学习这门课程吧。
