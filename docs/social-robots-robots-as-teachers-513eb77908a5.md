# 科学家为教室训练机器人

一组科学家已经在教室里测试了机器人。所谓“监督式渐进自主机器人能力”，简称 SPARC。在这个过程中，机器人由真人监督，应该能够自主学习。机器学习技术还没有准备好让机器人教授课堂行为。在发表于《科学机器人学》的研究中，科学家们控制了机器人及其动作。机器人存储早期的动作，然后向人类提出这些动作，人类然后确认或拒绝它们。动作执行得越频繁，机器人学习得越多。最终目标是机器人能够在没有人类帮助的情况下，自主地在教室里执行正确的动作。

# 这个机器人能够教 75 个孩子，对他们的表现产生了积极的影响

在这项研究中，使用了一个仿人 Nao 机器人。在 25 个单元和不到 3 个小时的互动后，他能够挽救社会行为，因此他能够教 75 个孩子。使用机器人的学生的表现提高了 10%到 30%。当然，当学生被机器人教授时，从种族和教育学的角度来看，这仍然需要考虑它有多有用。然而，可以假设这些测试不是为了好玩而进行的，在它背后有一个对未来的意图。
