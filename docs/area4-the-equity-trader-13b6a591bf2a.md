# 区域 4:股票交易员

> 原文：<https://medium.datadriveninvestor.com/area4-the-equity-trader-13b6a591bf2a?source=collection_archive---------35----------------------->

[![](img/32684b91b1d8ecb23dc49a9169177409.png)](http://www.track.datadriveninvestor.com/1B9E)

大公司找大项目。

他们的自动化解决方案总是大量推出，在软件上投资了数百万美元。

这是让你的业务更有效率的最好方法吗？

我认为各级工作都有巨大的改进空间。尤其是在大公司。在这一点上，这些改进可以是幕后的，在后台悄悄地工作。没有培训和维护成本。

让我们假设一个股票交易者。叫她莫里森。

要自动化初级股票交易员的整个工作流程，需要相当大的努力和非常大量的数据来训练多年来开发的最先进的人工智能。但是自动化 60%的工作流很容易通过脚本和后台进程的组合来完成。

莫里森日常工作的一大部分是“处理邮件”。他的团队将通过电子邮件接收来自不同公司的市场分析。任务是在“sink”电子邮件地址中找到这些电子邮件，并将它们合并成一份报告，供整个团队每周讨论。这个任务对计算机来说非常简单，不需要任何最先进的区块链机器学习。把它交给电脑，可以节省莫里森宝贵的认知空间，让他有时间在更有吸引力的任务中表达他们的创造力。这使得任务实际上消失了。它在后台自动发生。当所有数据都可用时，输出才出现在正确的位置。

典型工作流程的另一部分是查找股票信息。每个交易员都有自己喜欢的数据库/彭博终端功能等。他们查看自己的个人混合信息，每次点击通过相同的用户界面迷宫。这可以实现自动化，并放入一个仪表板中，从而节省时间。
在这种情况下，自动化提高了效率，简化了可重复的任务。当你一天做一百次事情时，每次节省 5 次点击可能是值得的。有时可能不是:

![](img/54c974512657e064bf0594316b9b5c10.png)

和任何好问题一样:“视情况而定”。
