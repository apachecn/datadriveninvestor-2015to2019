近几十年来，编程已经成为一种流行的爱好。随着紧凑型个人电脑、笔记本电脑和媒体的兴起，普通人掌握这项技能变得更加容易。这种技能不仅被具有工程或科学教育背景的团体所追求，在某些情况下也被音乐家、艺术家、会计或运动员所追求。

**1。这是一个全新的世界**

A whole new world (Cr: giphy.com)

当你说‘我做编程’的时候，这是一个非常宽泛和笼统的术语，用来描述你正在做的事情。编程已经发展成为一个有待探索的非常有趣的领域。学习编程语言是探索 web 应用程序、移动应用程序、数据科学家、人工智能(AI)、机器学习、游戏开发、虚拟现实(VR)、增强现实(AR)、IOT 等世界的第一步。

当你开始学习编程的时候，你接下来要学的东西是没有止境的。随着你继续深入挖掘这项技能，你会不断学习并取得小的进步。根据心理学研究，作为人类，当我们在追求的每一项技能上都取得进步时，我们会在情感上更快乐，对生活更满意。:)

**2。它训练你从零开始思考的能力**

(Cr : giphy.com)

程序员是如何创造软件的？不是靠魔法突然冒出来的。这需要想象力和循序渐进的思考。

当你学习如何创造一件产品时，你将学习如何通过想象和循序渐进的过程从无到有地创造一些东西。你心中有一个最终产品，你需要精心制定一个计划，每一步都是实现最终产品的渐进过程。就像一个画风景的艺术家，你会从普通的画布开始，每一笔，你都会不断加入元素，直到你的画实现了你的想象。

**3。它训练你自律和专注**

(Cr : giphy.com)

编程的本质不在于一个绝妙的想法或天才的大脑。但这是一种持续盯着屏幕几个小时，思考并寻找解决方案的能力。这里的主要技巧是集中注意力，避免拖延，直到你的问题/任务得到解决。众所周知，专注和自律的能力是可以训练的，这对我们的生活非常有益。通过学习编程，每一个问题/任务/项目都会训练你专注和自律。

**4。很洒脱**

(Cr : giphy.com)

是啊！它是免费的！你需要的只是坚持不懈的努力。随着媒体和互联网的进步，免费资源无处不在。在编程的每个领域都可以找到 YouTube 教程、博客和结构化课程。当你发现一个路障时，大量的程序员会从 StackOverflow 或 GitHub 帮助你。

这很容易。是啊！这很容易。编程不是受过特殊教育的特殊技能，也不是天才大脑。编程是解决问题的技能。它通过每天练习技能而进步。你所需要做的就是坚持每天在特定的时间内编码。非常无聊的工作不是吗？
