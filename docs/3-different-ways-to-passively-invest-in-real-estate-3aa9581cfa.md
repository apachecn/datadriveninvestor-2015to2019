# 3 种被动投资房地产的不同方式

> 原文：<https://medium.datadriveninvestor.com/3-different-ways-to-passively-invest-in-real-estate-3aa9581cfa?source=collection_archive---------51----------------------->

[![](img/dbeacb7f9e3ab18564d4a813c26fb4a9.png)](http://www.track.datadriveninvestor.com/DDI-Infograph-11-16)![](img/9165ac9de12670fd3ed2403241f97386.png)

房地产是一个伟大的投资工具，有很多优势投资辛迪加。我之前在我的一篇文章[与辛迪加一起投资的好处](https://www.ellieyogev.com/blog/the-significant-benefits-of-investing-alongside-a-syndicator)中写过被动投资辛迪加的好处。当想到被动的房地产投资时，大多数投资者会想到以有限合伙人的身份投资一笔交易，但还有更有创意的方式来投资房地产。

**1。借给辛迪加存款的钱**

在今天的市场上，大多数辛迪加需要写一张存款支票，根据市场和交易规模，存款金额可以在 10 万美元到 50 万美元之间。在竞争更加激烈的市场中，辛迪加需要“硬”支付部分或全部押金，这意味着押金可以退还，即使交易没有通过(有一些例外，如所有权或环境问题，有时融资意外情况是可以接受的)。被动投资者可以借出存款以换取股权或作为硬贷款人，在这种情况下，她/他将获得贷款利息。

**2。带上资产负债表**

一些辛迪加缺乏适当的资产负债表来提交要约，因为经纪人和卖方经常要求资金证明(“POF”)来确保辛迪加有购买的手段(特别是因为，为了获得贷款，一个人需要具有等于贷款金额的净值)。被动投资者可以因提供资产负债表和净值信息而获得少量股权。

**3。签贷款**

正如我前面提到的，为了获得商业贷款，贷款人要求借款人的净值等于贷款金额和 9-12 个月贷款支付的流动性。在这个定义中，净资产不包括个人住房的价值。大多数商业贷款是无追索权贷款，这意味着除了贷款所涉及的财产之外，贷款人不能收取任何抵押品。唯一的例外是一个坏男孩创业；根据这一例外，在辛迪加或委托人犯下某些列举的不良行为时，签署贷款的委托人要承担个人责任。

被动投资者可以签署贷款，作为普通合伙企业(GP)股权的关键本金(KP)，通常为 GP 的 5%-10%。作为一个被动的投资者，你可以避免贷款违约，但是，如果辛迪加犯了欺诈罪，你可能会发现自己要为全部贷款承担责任。因此，被动投资者要么只有在他们个人或专业上熟悉辛迪加的情况下才签署贷款，要么在决定成为金伯利进程之前花一些时间审查辛迪加。

在我以前的一篇文章中，我讨论了不用自己的钱投资房地产的各种方法，以及成为被动投资者的方法。成为房地产的被动投资者可能是一个聪明而安全的冒险，因为投资的方式不止一种；你可以发挥创造力，通过房地产投资找到许多积累财富的方法。
