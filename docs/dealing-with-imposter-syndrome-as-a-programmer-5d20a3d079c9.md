# 什么是冒名顶替综合症？

冒名顶替综合征可以被定义为尽管取得了明显的成功，但仍持续存在的一系列不充分的感觉。始终无法相信自己的成功是应得的，或者是通过自己的努力和技能取得的。

# 你问过自己这些简单的问题吗？

> ***我足够聪明成为一名程序员吗？***

很多人在成长过程中将编程视为天才之作。也许在学生时代和小学时代表现优异的人应该成为一名程序员。

![](img/ccf5bfc1971144c5a41709751639d6fd.png)

但是，事实是我从来没有想过成为一名程序员，甚至没有开始编程。直到我上了大学，我才开始考虑编程，这是一个非常艰难的开始。我认为与其说是天生的智力，不如说是野心和动力。

> 我属于这里吗？

相信我，刻板印象是胡说八道。你的长相和穿着并不重要，这些都不重要。你可能认为你可能会被你周围的人评判，而你可能会被一小群人评判，这些人绝对可以忽略不计。简而言之，没有人关心所有这些事情，人们只关心交给你的工作是否由你按时完成，你知道你在做什么。

> ***难道只是运气？***

我能在这里真的是因为我的努力和技能，还是仅仅是人们和我的运气帮助我实现了这个目标？每两个人都会问自己一个非常幼稚但非常普遍的问题。

# 编程中冒名顶替综合症盛行的原因？

> 软件开发每天都在发展。

每天都有大量的新技术和新东西进入软件开发领域，你需要跟上它的步伐以保持更新和不丢失。可能你刚学到一些东西，它在很短的时间内就更新了。

> ***程序员的天性***

你自己想想为什么会有冒名顶替综合症。答案在你:)

# 以新的方式思考如何克服冒名顶替综合症。

有几件事要记住，以帮助你与冒名顶替综合症。不舒服也要舒服。相信我，你永远不会成为你所在领域的专家。永远不要让自己失望，永远不要低估自己。不要拿自己和别人比较。
