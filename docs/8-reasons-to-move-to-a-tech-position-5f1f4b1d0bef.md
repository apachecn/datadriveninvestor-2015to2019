最近，我从在制药和临床研究行业工作了九年的工作岗位转到了一个更加科技化的岗位。

32 岁的时候，我决定思考我想要什么样的生活，尽管还在努力寻找，我还是决定回去学习 web 开发。以下是我选择 web 开发的原因以及我是如何做的。

1.  创造力 -网络开发让你以一种有趣和分析性的方式发挥创造力。它允许你同时使用大脑的逻辑部分和创造性部分。设计和实现一个网站需要逻辑编码技能以及前端设计技能。
2.  **工作机会**——虽然脸书的使用没有减少，但它的目标人群已经发生了变化。如今，企业开始重新投资动态网络应用和移动应用，并拥有 Instagram 和脸书账户。数字时代已经到来，随着人工智能的引入，工作和我们的工作方式将发生根本性的变化。卡斯特尔的网络社会似乎已经到来，技术升级似乎是一个合乎逻辑的举动，因为工作的所有方面都将很快以技术为基础。
3.  **自由**——尽管这无论如何都不是一件容易的工作，但如果你坚持下去并不断学习，你最终会达到可以为自己工作的地步。让你掌控自己的生活和时间，而不是某个愤怒的中层管理者。
4.  学习——科技行业的工作需要不断学习。你将永远不会停止学习和提高技能。你在不断提高你的技能，改进你的手艺。这让你永远不会感到停滞不前，给人一种你总是在朝一个方向前进的感觉。
5.  解决问题——决定进入一个以技术为基础的职业会教会你解决问题。它会教你把每件事都分解成小部分，然后在这些小部分上努力，直到每个部分加起来成为一个大部分。
6.  当你完成一个网页，甚至只是让一小段代码工作起来，这是令人难以置信的满足感。
7.  **信心**——学习任何种类的编码都会做两件事。开始时，它会让你觉得自己愚蠢和无能，但渐渐地，它会建立你的信心，因为你意识到你事实上可以做到，你足够聪明。
8.  谦逊——进入科技行业将教会你谦逊。要明白，总有更聪明、更有经验的人，但只要有毅力和毅力，你就能做到，你就能进步。

在第一学期奇迹般地通过了所有的考试和连续评估后，我决定冒一次大险。我辞掉了医药行业高薪的办公室工作。

我在第二学期集中精力完成了我所有的项目和考试，最近刚刚在一家顶级科技公司找到了一份基于网络的新工作。虽然这个职位不是我想要的，但绝对是朝着正确方向迈出的一步。

这段旅程教会我，如果你想要什么，你所要做的就是抓住机会，坚持不懈。别人成功而你认为自己没有成功的唯一原因是他们比你坚持得更久。

所以，如果你想离开，我会说去吧！
