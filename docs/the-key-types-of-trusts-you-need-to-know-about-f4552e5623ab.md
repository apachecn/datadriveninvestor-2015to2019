# 你需要了解的主要信托类型

当你想保护你的资产免受征税，并确保你的财富代代相传时，你将不可避免地遇到信托的想法。这是一个强大的金融工具，它允许你以合法和透明的方式保护你的资产，但这可能是一个复杂的过程。为了让你对可供选择的方案有新的认识，我们将对你需要了解的关键信任类型进行介绍。

**申报与结算**

信托的结构和目的将在由相关各方签署的正式书面文件中规定。您需要了解两种关键类型的文档。第一个是信托声明，这是一份由受托人单独签署的文件。它概述了他们将如何经营和管理信托，并不要求信托公司说出其持有的资产的来源。近年来，和解——或信托契约——变得越来越流行。它由托管人和受托人签署，并再次概述了信托将如何运行。

**可撤销信托**

顾名思义，可撤销信托是指托管人有权在其去世前的任何时候改变信托的结构。这包括更改或取消对受益人以及受益人本人的分配。对于相对不熟悉信托的高净值个人来说，可撤销信托是一种常见的选择，他们正在寻找一种非永久性的解决方案，让他们在一生中拥有一定程度的灵活性和自主权。然而，应该注意的是，托管人去世后，可撤销的信托自动变成不可撤销的，这意味着资产和分配不能再改变。

**不可撤销信托**

不可撤销信托是指任何不能由委托人自己撤销的信托。他们没有权力在任何时候转移资产、改变受益人或改变分配。然而，有很多方法可以改变，只要不是单方面的。所有这些方法通常都需要受托人的书面同意。受托人还可以选择缩短信托的期限，从而分配其持有的所有资产和资本。

**定息信托**

通常选择固定利息信托是因为这种简单的分配方式有助于解决由分配大量资产引起的政治问题。固定利息信托以预先确定的时间间隔支付固定金额的资本或利息，将有助于确保未来几代人不会受到大量财富涌入的影响，这些财富会动摇他们对金融资产的看法。然而，应该指出的是，由于其固定的性质，这些类型的信托在离婚诉讼中可以被视为婚姻财产。

**全权信托**

在全权信托的情况下，受托人始终对受益人分配的规模和频率拥有唯一和最终的发言权。他们的行为通常以托管人的意愿书为指导，该意愿书表明他们希望信托如何运行。重要的是要注意，受托人不受本文件的约束，只是在第一种情况下受其指导。这意味着，虽然托管人和其他相关方可能要求受托人以特定的方式行事，但受托人始终保持控制权。
